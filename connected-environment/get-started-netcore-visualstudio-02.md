# <a name="get-started-on-connected-environment-with-net-core-and-visual-studio"></a>Introducción a Connected Environment con .NET Core y Visual Studio

Paso anterior: [Instalar las herramientas](get-started-netcore-visualstudio-01.md)

## <a name="create-an-aspnet-web-app"></a>Crear una aplicación web ASP.NET
Cree un proyecto en Visual Studio 2017; actualmente, debe ser una **aplicación web ASP.NET Core**. Denomine el proyecto **webfrontend**.

![](images/NewProjectDialog1.png)

Seleccione la plantilla **Aplicación web (controlador de vista de modelos)** y asegúrese de que las dos listas desplegables en la parte superior del cuadro de diálogo tienen establecido como destino **.NET Core** y **ASP.NET Core 2.0**. Haga clic en **Aceptar** para crear el proyecto.

![](images/NewProjectDialog2.png)
