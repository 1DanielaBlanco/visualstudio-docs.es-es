# <a name="c1252"></a>C1252
Advertencia C1252: dependencia Circular o que faltan entre complementos: requiere GUID  
  
 La herramienta de análisis de código emite esta advertencia cuando se produce un error interno con dependencias de complemento, no en el código que se va a analizar.
