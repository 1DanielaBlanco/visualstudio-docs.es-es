# <a name="c1254"></a>C1254
Advertencia C1254: error de coincidencia de versión de complemento: versión no coincide con la versión del controlador PREfast  
  
 La herramienta de análisis de código emite esta advertencia cuando se produce un error interno con la versión del complemento, no en el código que se va a analizar.
