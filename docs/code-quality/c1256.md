# <a name="c1256"></a>C1256
Advertencia C1256: PCH debe ser tanto se escriben y se leen.  
  
 La herramienta de análisis de código emite esta advertencia cuando se produce un error interno en la herramienta, pero no en el código que se va a analizar.
