# C1256
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C1256: PCH debe ser escrito y leído  
  
 La herramienta de análisis de código señala esta advertencia cuando hay un error interno en la herramienta, no en el código que es analizado.
