# <a name="c26464-useconstpointerforelements"></a>C26464 USE_CONST_POINTER_FOR_ELEMENTS
  Los valores que se apunta a elementos de matriz '% matriz %' se asignan solo una vez, marque los elementos como puntero a `const`. Consulte [C++ Core Guidelines con.4](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#con4-use-const-to-define-objects-with-values-that-do-not-change-after-construction).
