# <a name="c26483-staticindexoutofrange"></a>C26483 STATIC_INDEX_OUT_OF_RANGE

Valor de % de valor está fuera de los límites (0, enlazadas %) de la variable '% variable %'. Solo el índice de matrices mediante expresiones constantes que están dentro de los límites de la matriz. Vea [Bounds.2 de directrices de núcleo de C++](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-bounds)
