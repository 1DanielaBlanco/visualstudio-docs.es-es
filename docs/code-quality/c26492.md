# <a name="c26492-noconstcast"></a>C26492 NO_CONST_CAST

No use `const_cast` para desechar `const`. Consulte [C++ Core Guidelines tipo.3](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-type).
