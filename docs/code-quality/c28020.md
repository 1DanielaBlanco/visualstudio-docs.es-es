# C28020
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28020: La expresión \<expresión\> no es verdadera en esta llamada  
  
 Esta advertencia se muestra cuando la expresión \_Satisfies\_ indicada no es verdadera.  Con frecuencia esto indica un parámetro incorrecto.  
  
 Si esto ocurre en una declaración de función, las anotaciones indican una condición imposible.
