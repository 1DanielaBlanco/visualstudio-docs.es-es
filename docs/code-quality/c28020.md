# <a name="c28020"></a>C28020
Advertencia C28020: la expresión \<expr > no es cierto en esta llamada  
  
 Esta advertencia se muestra cuando el _Satisfies\_ expresión enumerados no es true. Con frecuencia, esto indica un parámetro incorrecto.  
  
 Si esto ocurre en una declaración de función, las anotaciones indican una condición imposible.
