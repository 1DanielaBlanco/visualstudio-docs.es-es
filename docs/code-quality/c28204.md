# C28204
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28204: \<función\> : solo se anota una de estas sobrecargas y la que aparece en \<nombre de archivo\>\(\<línea\>\) para \<nombre de parámetro\>: se deben anotar ambas o ninguna  
  
 La advertencia se muestra cuando hay un error en las anotaciones.
