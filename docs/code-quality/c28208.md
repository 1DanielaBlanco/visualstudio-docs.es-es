# <a name="c28208"></a>C28208
Advertencia C28208: función \<función > se ha definido con una lista de parámetros diferentes en \<archivo > (\<línea >). Algunas herramientas de análisis produzca resultados incorrectos  
  
 Esta advertencia se muestra cuando definición conocidos de una función no coincide con otra aparición.
