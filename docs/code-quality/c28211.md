# C28211
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28211: Se esperaba un nombre de contexto estático para SAL\_context  
  
 Esta advertencia indica que el operando a la anotación `_Static_context_` debe ser el nombre de un contexto de herramienta definida.  Esta advertencia nunca debe aparecer en el uso normal.
