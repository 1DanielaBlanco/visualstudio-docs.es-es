# <a name="c28213"></a>C28213
Advertencia C28213: el `_Use_decl_annotations_` anotación se debe usar para hacer referencia, sin modificaciones, una declaración anterior. \<¿Por qué >  
  
 Esta advertencia se muestra cuando se hace referencia a una declaración anterior sin necesaria `_Use_decl_annotations_` anotación.
