# C28213
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28213: La anotación `_Use_decl_annotations_` debe utilizarse para hacer referencia, sin modificación, a una declaración previa \<por qué\>.  
  
 Esta advertencia aparece cuando una declaración previa es referenciada sin la anotación `_Use_decl_annotations_` requerida.
