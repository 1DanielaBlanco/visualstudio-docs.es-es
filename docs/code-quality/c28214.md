# C28214
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28214: Los nombres de los parámetros de atributo deben ser p1...p9  
  
 Esta advertencia indica que al construir una declaración de anotación, los nombres de parámetro están limitados a p1…p9.  Esta advertencia nunca debe aparecer en el uso normal.
