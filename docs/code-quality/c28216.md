# <a name="c28216"></a>C28216
Advertencia C28216: el _Check_return\_ anotación solo se aplica a las condiciones posteriores para el parámetro de función específica.  
  
 El `_Check_return_` anotación se ha aplicado en un contexto distinto de `__post`; puede que necesite un `__post` (o `__drv_out`) modificador, o bien se puede colocar incorrectamente.
