# C28216
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C28216: La anotación \_Check\_return\_ sólo se aplica a las post\-condiciones para el parámetro de la función específica.  
  
 La anotación `_Check_return_` se aplicó en un contexto distinto `__post`; puede necesitar un modificador `__post` \(o `__drv_out`\), o puede ser colocada incorrectamente.
