# <a name="c28217"></a>C28217
Advertencia C28217: para la función, el número de parámetros para la anotación no coincide con encontrado en el archivo  
  
 Las anotaciones en la línea actual y en la línea en el mensaje son incoherentes. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en el uso normal.
