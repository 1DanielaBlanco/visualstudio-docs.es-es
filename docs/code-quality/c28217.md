# C28217
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C28217: Para la función, el número de parámetros de la anotación no coincide con el encontrado en el archivo  
  
 Las anotaciones en la línea actual y en la línea del mensaje son incoherentes.  Esto no debería ser posible si las macros estándar se utilizan para las anotaciones; esta advertencia no se espera en un uso normal.
