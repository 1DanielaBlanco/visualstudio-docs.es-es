# C28219
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28219: Miembro de enumeración esperado para la anotación del parámetro en la anotación  
  
 Se espera un parámetro para una anotación para ser miembro del tipo denominado `enum` y algún otro símbolo encontrado; utilice un miembro de tipo `enum`.  Normalmente indica una macro de anotación incorrectamente codificada.
