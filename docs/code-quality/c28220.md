# <a name="c28220"></a>C28220
Advertencia C28220: esperaba una expresión de entero para el parámetro de la anotación  
  
 Esta advertencia indica que se espera un parámetro a una anotación a ser una expresión de entero y se encontró algún otro tipo. Esto normalmente indica que una macro de anotación de código incorrecto.
