# C28221
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28221: Expresión de cadena esperada para el parámetro en la anotación  
  
 Esta advertencia indica que se espera que un parámetro para una anotación sea una cadena y se encontró otro tipo.  Esto normalmente indica una macro de anotación incorrectamente codificada y no se espera ver en un uso normal.
