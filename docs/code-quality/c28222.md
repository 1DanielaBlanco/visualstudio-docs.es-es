# C28222
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia 28222: \_Yes\_, \_No\_, o \_Maybe\_ esperado para la anotación  
  
 Esta advertencia indica que se espera que un parámetro a una anotación sea uno de los símbolos `_Yes_`, `_No_`, o `_Maybe_` y que, al contrario, se encontró otro símbolo.  Normalmente indica una macro de anotación incorrectamente codificada.
