# <a name="c28223"></a>C28223
Advertencia C28223: No se encontró el token o identificador para la anotación, parámetro

 Esta advertencia indica que se espera un parámetro a una anotación a ser un identificador, y se encontró otro símbolo. Esto suele indicar una macro de anotación de código incorrecto. El uso de palabras clave de C o C++ en esta posición producirá este error.
