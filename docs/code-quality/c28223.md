# C28223
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28223: No se encontró el token o identificador esperado para el parámetro de la anotación  
  
 Esta advertencia indica que se espera que el parámetro para una anotación sea un identificador y cualquier otro símbolo encontrado.  Normalmente indica una macro de anotación incorrectamente codificada.  El uso de las palabras clave de C o C\+\+ en esta posición producirá este error.
