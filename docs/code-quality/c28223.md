# <a name="c28223"></a>C28223
Advertencia C28223: No se encontró Token o identificador para la anotación, parámetro

 Esta advertencia indica que se espera un parámetro a una anotación para ser un identificador, y se encontró algún otro símbolo. Esto normalmente indica que una macro de anotación de código incorrecto. El uso de palabras clave de C o C++ en esta posición provocará este error.
