# C28224
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28224: La anotación requiere parámetros  
  
 Esta advertencia indica que la anotación denominada se utiliza sin parámetros y al menos un parámetro es obligatorio.  Esto no debería ser posible si las macros estándar se utilizan para las anotaciones; esta advertencia no se espera en un uso normal.
