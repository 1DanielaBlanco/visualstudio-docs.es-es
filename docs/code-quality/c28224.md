# <a name="c28224"></a>C28224
Advertencia C28224: anotación requiere parámetros  
  
 Esta advertencia indica que la anotación con nombre se usa sin parámetros y se requiere al menos un parámetro. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en el uso normal.
