# <a name="c28225"></a>C28225
Advertencia C28225: no se encontró el número correcto de parámetros necesarios en la anotación  
  
 Esta advertencia indica que la anotación con nombre se usa con el número incorrecto de parámetros. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en uso típico.
