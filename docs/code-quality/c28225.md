# C28225
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28225: No se encontró el número correcto de parámetros requeridos en la anotación  
  
 Esta advertencia indica que la anotación denominada se utiliza con el número incorrecto de parámetros.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
