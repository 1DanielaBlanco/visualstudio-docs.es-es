# C28227
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28227: La anotación no puede ser un PrimOp \(consulte la declaración anterior\)  
  
 Esta advertencia indica que la anotación denominada se declara como anotación ordinaria y también se declaró previamente como PrimOp.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
