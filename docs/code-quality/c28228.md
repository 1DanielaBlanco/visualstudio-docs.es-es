# C28228
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28228: Parámetro de anotación: no se puede usar un tipo en las anotaciones  
  
 Esta advertencia indica que un parámetro es de un tipo no admitido.  Las anotaciones sólo pueden utilizar como parámetros un conjunto limitado de tipos.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
