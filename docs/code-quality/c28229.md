# <a name="c28229"></a>C28229
Advertencia C28229: anotación no admite parámetros

 Esta advertencia indica que se utilizó una anotación con un parámetro cuando la anotación se ha declarado sin parámetros. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en uso típico.
