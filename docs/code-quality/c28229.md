# C28229
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28229: La anotación no admite parámetros  
  
 Esta advertencia indica que una anotación con un parámetro se utilizó cuando la anotación se declara sin parámetros.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
