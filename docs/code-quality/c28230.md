# C28230
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28230: El tipo de parámetro no tiene ningún miembro.  
  
 Esta advertencia indica que un argumento para una anotación intenta obtener acceso a `struct`, `class`, o a `union` y no existe el miembro denominado.
