# C28231
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28231: La anotación solo es válida en la matriz  
  
 Esta advertencia indica que un argumento a una anotación debe ser un array y se encontró algún otro tipo.
