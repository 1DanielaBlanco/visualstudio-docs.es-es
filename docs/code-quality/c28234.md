# C28234
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C28234: La expresión \_AT\_ no se aplica a la función actual  
  
 Esta advertencia indica que el valor de una expresión `_At_` no identifica un objeto accesible.
