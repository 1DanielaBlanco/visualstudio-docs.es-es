# C28236
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28236: La anotación no se puede usar en una expresión  
  
 Esta advertencia indica que se intentó utilizar una función declarada como anotación en un contexto de expresión.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
