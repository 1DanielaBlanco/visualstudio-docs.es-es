# <a name="c28236"></a>C28236
Advertencia C28236: no se puede usar la anotación en una expresión  
  
 Esta advertencia indica que se ha intentado usar una función declarada como una anotación en el contexto de una expresión. Esto no debería ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en uso típico.
