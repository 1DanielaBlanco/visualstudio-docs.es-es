# C28239
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C28239: La anotación en el parámetro tiene tanto un elemento value como stringValue o longValue, además de paramn\=xxx.  Use solamente paramn\=xxx.  
  
 Un error interno ha ocurrido en el archivo de modelo PREfast.  La advertencia no debe aparecer en el uso común.
