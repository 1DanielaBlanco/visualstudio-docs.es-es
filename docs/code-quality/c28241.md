# <a name="c28241"></a>C28241
Advertencia C28241: La anotación para la función del parámetro no se reconoce

 Se usó un nombre de anotación no reconocido. Esto no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra un uso típico.
