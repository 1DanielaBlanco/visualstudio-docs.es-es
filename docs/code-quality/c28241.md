# C28241
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28241: No se reconoce la anotación para la función en el parámetro  
  
 Se utilizó un nombre desconocido de anotación.  Esto no debería ser posible si los macros estándar se utilizan para las anotaciones; no se espera ver esta advertencia en el uso normal.
