# <a name="c28244"></a>C28244
Advertencia C28244: la anotación de la función tiene una anotación de parámetro no analizable/externo  
  
 Actualmente no debe ser posible si se utilizan las macros estándares para las anotaciones; Esta advertencia no se espera para ser vistas en uso típico.
