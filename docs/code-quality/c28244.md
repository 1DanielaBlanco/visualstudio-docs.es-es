# C28244
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28244: La anotación para la función tiene un parámetro o anotación externa que no se puede analizar  
  
 Esto no debería ser actualmente posible si los macros estándar se utilizan para las anotaciones; esta advertencia no se espera en un uso normal.
