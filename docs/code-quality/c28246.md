# C28246
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28246: La anotación para la función '\<nombre\>' \- parámetro '\<parámetro\>' no coincide con el tipo de parámetro  
  
 Un operador \_\_deref fue aplicado a un tipo no puntero al crear una anotación.
