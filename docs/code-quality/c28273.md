# C28273
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28273: Por función, las pistas no son coherentes con la declaración de función  
  
 Esta advertencia indica un error en las anotaciones, no en el código que se está analizando.  Las anotaciones que aparecen en una definición de función son incoherentes con las que aparecen en una declaración.  Las dos anotaciones deben resolverse de forma que coincidan.
