# C28275
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28275: El parámetro para \_Macro\_value\_ es null  
  
 La advertencia indica que existe un error interno en el archivo modelo, no en el código que es analizado.  La función de *macroValue* se llamó sin un parámetro.
