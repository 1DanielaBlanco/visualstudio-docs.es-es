# C28279
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28279: Para el símbolo, se encontró un elemento 'begin' sin el elemento 'end' correspondiente  
  
 El lenguaje de anotación permite una construcción de inicio y final \(`{` y `}` en C\) y el emparejamiento se ha desequilibrado.  Esta situación puede evitarse si se utilizan macros.
