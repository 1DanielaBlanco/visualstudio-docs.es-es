# <a name="c28279"></a>C28279
Advertencia C28279: para el símbolo, un 'begin' se encontró sin un 'end' coincidente  
  
 El lenguaje de anotación admite un begin y end (`{` y `}` en C) ha perdido desequilibrada la construcción y el emparejamiento. Esta situación se puede evitar si se utilizan las macros.
