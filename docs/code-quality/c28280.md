# C28280
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28280: Para el símbolo, se encontró un elemento 'end' sin el elemento 'begin' correspondiente  
  
 El lenguaje de anotación admite una construcción de inicio y final \(`{` y `}` en C\) y el emparejamiento se ha desequilibrado.  Esta situación puede evitarse si se utilizan macros.
