# C28283
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28283: Para el símbolo, no se admite aún la especificación de tamaño indicada  
  
 La advertencia indica que la anotación de “centinela” de la especificación de tamaño recibió un valor distinto de cero.  Esencialmente, el llamador intentó establecer que la cadena se cancele por un carácter distinto al cero binario.
