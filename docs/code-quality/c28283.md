# <a name="c28283"></a>C28283
Advertencia C28283: para el símbolo, la especificación de tamaño especificado aún no se admite  
  
 La advertencia indica que la anotación sentinel"especificación de tamaño" recibió un valor distinto de cero. En esencia, el autor de llamada intentó decir que la cadena termina con un carácter distinto de cero binario.
