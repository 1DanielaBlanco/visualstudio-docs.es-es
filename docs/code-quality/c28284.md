# C28284
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28284: Para el símbolo, los predicados no se admiten actualmente en símbolos que no son de función  
  
 Esta advertencia indica que una anotación condicional \(predicado, `__drv_when`\) se ha encontrado en algún elemento distinto de una función.
