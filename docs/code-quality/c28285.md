# C28285
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28285: Para la función, error de sintaxis en el parámetro  
  
 La herramienta de análisis de código lanza esta advertencia cuando se encuentra un probable error en el archivo del modelo.  Algunos errores de archivos de código fuente también pueden producir estos errores.
