# <a name="c28285"></a>C28285
Advertencia C28285: para la función, error de sintaxis en el parámetro  
  
 La herramienta de análisis de código informa de esta advertencia cuando se produce un error probables en el archivo de modelo. Algunos errores de archivo de origen también pueden producir estos errores.
