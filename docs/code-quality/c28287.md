# <a name="c28287"></a>C28287
Advertencia C28287: para la función, Error de sintaxis en _superior a\_anotación () (nombre de parámetro no reconocido)

 La herramienta de análisis de código informa de esta advertencia cuando el `SAL_at` (`__drv_at`) se utiliza la anotación y no se puede interpretar la expresión del parámetro en el contexto actual. Esto podría incluir usando un parámetro mal escrito o nombre de miembro o una palabra mal escrita de "return" o "this" palabras clave.
