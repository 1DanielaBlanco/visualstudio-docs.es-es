# C28289
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

advertencia C28289: La función: ReadableTo o WritableTo no tenía un parámetro especificado por un límite  
  
 La herramienta de análisis de código señala esta advertencia cuando la anotación de funciones de parámetros se codifica erróneamente de la manera indicada.
