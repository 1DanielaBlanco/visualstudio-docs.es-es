# C28290
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28290: La anotación de la función contiene más valores External que el número real de parámetros  
  
 La herramienta de Análisis de código señala esta advertencia cuando la anotación para la función contiene más Externos que el número real de parámetros.
