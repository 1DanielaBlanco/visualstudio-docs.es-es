# <a name="c28291"></a>C28291
Advertencia C28291: Post null/notnull en deref nivel 0 no tiene sentido para la función \<x > en param \<número >  
  
 La herramienta de análisis de código informa de esta advertencia cuando se especifica la condición posterior de un parámetro de nivel cero desreferencia tener una propiedad es null y no null. Este error se produce porque se desreferencia un valor en el nivel cero no se puede cambiar.
