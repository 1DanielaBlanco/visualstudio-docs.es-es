# C28301
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28301: Ninguna anotación para la primera declaración de \<función\>.\<nota1\> consulte \<nombre de archivo\>\(\<línea\>\). \<nota2\>  
  
 Esta advertencia se muestra cuando las anotaciones no se encontraron en la primera declaración de una función dada.
