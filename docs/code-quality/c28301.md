# <a name="c28301"></a>C28301
Advertencia C28301: no hay ninguna anotación para la primera declaración de \<función >.\< note1 > vea \<nombre de archivo > (\<línea >). \<Nota 2 >

 Esta advertencia se muestra cuando no se encontraron las anotaciones en la primera declaración de una función determinada.
