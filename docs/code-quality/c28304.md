# <a name="c28304"></a>C28304
Advertencia C28304: parámetro de referencia de C++ para < nombre_parámetro >, un colocado `_Notref_` operador se encontró aplicado al \<token >.  
  
 El `_Notref_` operador solo debe utilizarse en circunstancias especiales que implica parámetros de referencia de C++ y solo en macros proporcionados por el sistema. Debe ir seguido inmediatamente por un `_Deref_` operador o una anotación funcional.
