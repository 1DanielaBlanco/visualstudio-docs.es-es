# <a name="c28311"></a>C28311
Advertencia 28311: la anotación de < annotation_name > en \<función > \<parámetro > es una versión obsoletas de SAL.

 La anotación es una versión antigua y debe actualizarse en el equivalente de [SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Esta advertencia no se genera si se ha emitido una advertencia de anotación incoherente anterior y se notifica una sola vez por declaración. Inspeccione el resto de la declaración para ver si hay más lenguaje SAL obsoleto.
