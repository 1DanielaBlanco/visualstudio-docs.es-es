# C28351
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C28351: La anotación \<anotación\> describe dónde no se puede usar un valor dinámico \(una variable\) en la condición  
  
 Esta advertencia se reporta normalmente cuando se aplica una anotación donde está siendo inspeccionado el tipo C\/C\+\+.  
  
 La expresión en `_When_` debe evaluarse como una constante.  `_When_` se omite.
