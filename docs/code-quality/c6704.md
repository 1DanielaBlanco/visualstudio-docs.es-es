# C6704
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Advertencia C6704: Error de expresión de anotación inesperado: \<anotación\> \[\<por qué\>\]  
  
 La advertencia se muestra cuando hay un error en las anotaciones.
