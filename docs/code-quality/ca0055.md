# <a name="ca0055"></a>CA0055
No se pudo cargar el archivo:\<ruta de acceso >

 La herramienta de análisis de código no pudo cargar el archivo de destino que se especificó para el análisis. Esto ocurre si no se encuentra el archivo o se denegó el acceso al archivo. Normalmente, esto se produce si el usuario decide reemplazar la propiedad de ensamblado de entrada en [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)], o invoca manualmente fxcopcmd.exe.
