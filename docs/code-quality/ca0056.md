# <a name="ca0056"></a>CA0056
Un archivo de proyecto tiene una versión incorrecta de la herramienta de análisis.  
  
 Esta advertencia se produce si FxCopCmd.exe se invoca manualmente con un archivo de proyecto que tiene una versión de herramientas de análisis de código incorrecto.
