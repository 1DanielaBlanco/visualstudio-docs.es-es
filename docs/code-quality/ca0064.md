# <a name="ca0064"></a>CA0064

Ningún análisis se realizan porque el conjunto de reglas especificado no contiene ninguna regla de FxCop.

Esta advertencia puede producirse en una de las siguientes situaciones:

- Si se produce esta advertencia junto con advertencias de CA0063, se produjo un error al cargar el archivo de conjunto de reglas. Para obtener más información, consulte el [CA0063](ca0063.md) documentación.

- Si se produce este error junto con advertencias de CA0062, lo más probable es que ese análisis de código pudo encontrar el ensamblado o ensamblados que contienen las reglas especificadas por el conjunto de reglas. Para obtener más información, consulte el [CA0062](ca0062.md) documentación.

- En caso contrario, esta advertencia suele ocurre cuando el conjunto de reglas está vacío o se han deshabilitado todas las reglas habilitadas en un conjunto de reglas secundarios. Usar Visual Studio [editor de conjunto de reglas](../code-quality/working-in-the-code-analysis-rule-set-editor.md) para habilitar algunas reglas en el conjunto de reglas.
