# <a name="ca0503"></a>CA0503
La propiedad está en desuso. Utilice la propiedad reemplaza  
  
 La advertencia CA0503 se genera cuando una propiedad de análisis de código desusada contiene un valor en el archivo de configuración de compilación o proyecto. Para resolver esta advertencia, utilice la propiedad especificada que la sustituye.
