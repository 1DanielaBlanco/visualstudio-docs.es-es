# <a name="ca0504"></a>CA0504
Se omitió el directorio de la regla porque no existe  
  
 Se genera la advertencia CA504 cuando se especifica un directorio en el **CodeAnalysisRuleDirectories** no se encontró la propiedad en un archivo de configuración de proyecto o de la compilación. Para resolver este problema, escriba una ruta de acceso de directorio válido.
