# <a name="ca0505"></a>CA0505

La propiedad desusada se omitirá porque se define la propiedad que la sustituye.

Advertencia: CA505 se genera cuando tanto las propiedades de análisis de código en desuso y reemplazo tienen valores en un proyecto o archivo de configuración de compilación. Para resolver este problema, quite la propiedad en desuso.
