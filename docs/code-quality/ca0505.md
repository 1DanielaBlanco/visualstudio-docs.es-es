# <a name="ca0505"></a>CA0505
La propiedad desusada se omitirá porque se define la propiedad que la sustituye.  
  
 Advertencia: CA505 se genera cuando tanto en desuso especificado como propiedades de análisis de código sustituye contienen valores en un proyecto o archivo de configuración de compilación. Para resolver este problema, quite la propiedad desusada.
