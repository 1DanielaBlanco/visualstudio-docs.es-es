# Medir la complejidad y el mantenimiento del c&#243;digo administrado
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

La complejidad incrementada de las aplicaciones de software modernas también aumenta la dificultad de hacer que el código sea confiable y fácil de mantener.  En años recientes, se han desarrollado muchas mediciones de software, conocidas como métricas de código, que pueden ayudar a los programadores a saber dónde se debe revisar el código o realizar más pruebas.  
  
 Los programadores pueden utilizar [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] para generar datos de métricas de código con el objeto de medir la complejidad y mantenimiento del código administrado.  Los datos de las métricas de código se pueden generar para una solución completa o para un proyecto único.  
  
 En esta sección se describe cómo generar y utilizar las métricas de código en [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
