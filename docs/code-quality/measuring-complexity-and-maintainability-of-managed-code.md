# <a name="measuring-complexity-and-maintainability-of-managed-code"></a>Medir la complejidad y el mantenimiento del código administrado
La mayor complejidad de las aplicaciones de software modernas también aumenta la dificultad de hacer que el código de confianza y fácil de mantener. En los últimos años, muchas medidas de software, conocidas como métricas de código, se han desarrollado que puede ayudar a los programadores a comprender que su código tiene repetición del trabajo o realizar más pruebas.  
  
 Los desarrolladores pueden usar [!INCLUDE[vsprvsts](../code-quality/includes/vsprvsts_md.md)] para generar datos de métricas de código que medir la complejidad y el mantenimiento del código administrado. Datos de métricas de código se pueden generar para una solución completa o un proyecto único.  
  
 Esta sección describe cómo generar y usar las métricas de código en [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
