# Suprimir advertencias mediante el atributo SuppressMessage
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Con frecuencia resulta útil indicar que la advertencia no es aplicable, para que los miembros del equipo sepan que el código se ha revisado y que se ha determinado que la advertencia se debe suprimir.  La supresión en el código fuente \(ISS\) permite al desarrollador colocar el atributo que suprime una advertencia cerca de la ubicación que ha generado la advertencia.  Puede agregar directamente el atributo de ISS al archivo de código fuente o puede utilizar el menú contextual del IDE de [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].
