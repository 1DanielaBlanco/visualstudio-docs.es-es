# No se pudo recuperar informaci&#243;n de esquema del objeto de base de datos &lt;nombre de objeto&gt;
Este mensaje suele aparecer cuando un objeto del **Explorador de servidores\/Explorador de bases de datos** se copia en el Portapapeles, se elimina de la base de datos y, a continuación, se pega en el diseñador.Dado que ya no existe el objeto de base de datos, aparece este mensaje.  
  
### Para corregir este error  
  
-   Haga clic en **Aceptar** para descartar el cuadro de diálogo.
