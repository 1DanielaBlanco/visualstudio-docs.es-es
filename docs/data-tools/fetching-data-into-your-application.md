# Buscar datos en la aplicaci&#243;n
La carga de datos en la aplicación describe el proceso de capturar los datos de un origen de datos, como una base de datos o servicio, y rellenando los objetos del modelo de datos de una aplicación.  En los siguientes temas se describe cómo capturar datos y cargarlos en los modelos de datos disponibles.
