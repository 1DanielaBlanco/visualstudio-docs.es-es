# C&#243;mo: Editar consultas de TableAdapter
Puede editar las consultas del TableAdapter con [Asistente para la configuración de consultas de TableAdapter](../data-tools/editing-tableadapters.md) en el **Diseñador de DataSet**.  Debe modificar una consulta de TableAdapter cuando ya no satisfaga las necesidades de la aplicación.  \(Además, puede crear consultas adicionales en TableAdapter.  Para obtener más información sobre cómo agregar nuevas consultas, vea [Cómo: Crear consultas de TableAdapter](../data-tools/how-to-create-tableadapter-queries.md).\)  
  
> [!NOTE]
>  Si se abre [Asistente para la configuración de TableAdapter](../Topic/TableAdapter%20Configuration%20Wizard.md) en lugar del **Asistente para la configuración de consultas de TableAdapter**, es posible que haya seleccionado la consulta `Fill` principal del objeto TableAdapter y ninguna de las consultas adicionales del TableAdapter.  Para obtener información sobre cómo editar la consulta `Fill` principal del TableAdapter, vea [Cómo: Editar TableAdapters](../Topic/How%20to:%20Edit%20TableAdapters.md).  
  
 ![TableAdapter con múltiples consultas](../data-tools/media/tableadapter.gif "TableAdapter")  
  
### Para editar una consulta de TableAdapter  
  
1.  Abra el conjunto de datos en el **Diseñador de Dataset**.  Para obtener más información, vea [Cómo: Abrir un objeto Dataset en el Diseñador de Dataset](../Topic/How%20to:%20Open%20a%20Dataset%20in%20the%20Dataset%20Designer.md).  
  
2.  Seleccione el TableAdapter que desee modificar.  
  
3.  Haga clic con el botón secundario en la consulta del TableAdapter y seleccione **Configurar**.  
  
     Se abre el **Asistente para la configuración de consultas de TableAdapter**, preparado para que se modifique la consulta o el procedimiento almacenado de esa consulta.  
  
4.  Finalice el **Asistente para la configuración de consultas de TableAdapter** con los cambios deseados.  Para obtener más información, vea [Asistente para la configuración de consultas de TableAdapter](../data-tools/editing-tableadapters.md).
