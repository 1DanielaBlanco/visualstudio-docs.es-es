> [!NOTE]
>  En [!INCLUDE[vs_dev12](../../extensibility/includes/vs_dev12_md.md)], puede conectarse a una base de datos de SQL Server desde el **Explorador de objetos de SQL Server** o **las conexiones de datos** nodo **Explorador de servidores**. **Explorador de objetos de SQL Server** proporciona una vista similar a SQL Server Management Studio. Para abrir este explorador desde la barra de menús, elija **vista**, **Explorador de objetos de SQL Server**. Para obtener más información, consulte [desarrollo de base de datos conectadas](http://go.microsoft.com/fwlink/?LinkId=233646).
