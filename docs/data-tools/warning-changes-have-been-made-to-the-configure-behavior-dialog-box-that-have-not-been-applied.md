# <a name="warning-changes-have-been-made-to-the-configure-behavior-dialog-box-that-have-not-been-applied"></a>Advertencia: Se realizaron cambios en el cuadro de diálogo Configurar comportamiento que no se hayan aplicado.

Advertencia. Se han realizado cambios en el cuadro de diálogo Configurar comportamiento que no se han aplicado. ¿Desea aplicar los cambios?

El **configurar comportamiento** cuadro de diálogo le permite configurar el comportamiento de Insert, Update y Delete para todas las clases disponibles. Este mensaje aparece cuando se selecciona un nuevo **clase** y **comportamiento** combinación y el cambio anterior no ha se ha aplicado.

## <a name="to-apply-the-change-and-continue"></a>Para aplicar el cambio y continuar

- Haga clic en **Sí** para aplicar el cambio a la seleccionada **clase** y **comportamiento**.

## <a name="to-cancel-the-previous-change-and-continue"></a>Para cancelar el cambio anterior y continuar

- Haga clic en **No** para cancelar el cambio seleccionada **clase** y **comportamiento**.
