# IDiaEnumFrameData::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Restaura una secuencia de enumeración al principio.  
  
## Sintaxis  
  
```  
HRESULT Reset();  
```  
  
## Valor devuelto  
 Devuelva `S_OK`.
