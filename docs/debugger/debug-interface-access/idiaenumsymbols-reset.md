# IDiaEnumSymbols::Reset
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Restaura una secuencia de enumeración al principio.  
  
## Sintaxis  
  
```cpp#  
HRESULT Reset ();  
```  
  
## Valor devuelto  
 Devuelva `S_OK`.
