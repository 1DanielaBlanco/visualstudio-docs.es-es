# <a name="idiaenumtablesreset"></a>IDiaEnumTables::Reset
Restablece una secuencia de enumeración al principio.  
  
## <a name="syntax"></a>Sintaxis  
  
```C++  
HRESULT Reset ();  
```  
  
## <a name="return-value"></a>Valor devuelto  
 Devuelve `S_OK`.
