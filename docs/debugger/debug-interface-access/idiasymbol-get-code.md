# <a name="idiasymbolgetcode"></a>IDiaSymbol::get_code
Recupera una marca que especifica si el símbolo hace referencia a una dirección de código.  
  
## <a name="syntax"></a>Sintaxis  
  
```C++  
HRESULT get_code (   
   BOOL* pRetVal  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pRetVal`  
 [out] Devuelve `TRUE` si el símbolo hace referencia a una dirección de código, en caso contrario, devuelve `FALSE`.  
  
## <a name="return-value"></a>Valor devuelto  
 Si se realiza correctamente, devuelve `S_OK`; en caso contrario, devuelve `S_FALSE` o código de error.  
  
> [!NOTE]
>  Un valor devuelto de `S_FALSE` significa que la propiedad no está disponible para el símbolo.
