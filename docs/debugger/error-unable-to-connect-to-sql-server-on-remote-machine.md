# Error: No se puede establecer una conexi&#243;n con SQL Server en el equipo remoto
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

No se puede conectar a SQL Server en el equipo remoto *name*.  Acceso denegado.  Compruebe que ha instalado el depurador remoto en él.  Si el equipo remoto no está en ningún dominio o si Visual Studio se ejecuta como cuenta local, el equipo remoto debe tener una cuenta con el mismo nombre de usuario y la misma contraseña que los de la cuenta local.  
  
### Para corregir este error  
  
-   Vea [Depuración remota](../debugger/remote-debugging.md).
