# Cuadro de di&#225;logo Fuente (Ayuda de Microsoft Spy++)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use este cuadro de diálogo para configurar la fuente de caracteres que se usa en una ventana de [vistas de Spy\+\+](../debugger/spy-increment-views.md).  Para mostrar este cuadro de diálogo, expanda el menú **Ver** y elija **Fuente**.  
  
 Están disponibles los siguientes parámetros de configuración:  
  
 **Fuente**  
 Fuente de caracteres de la ventana de vistas de Spy\+\+ que tiene el foco.  
  
 **Estilo de fuente**  
 Estilo de fuente de la ventana de vistas de Spy\+\+ que tiene el foco.  
  
 **Size**  
 Tamaño de fuente de la ventana de vistas de Spy\+\+ que tiene el foco.  
  
 **Guardar como predeterminada**  
 Guarda la configuración de fuente actual como preferencia al abrir las vistas de Spy\+\+.
