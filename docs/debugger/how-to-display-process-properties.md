# How to: Display Process Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Para obtener más información sobre un proceso que se muestra en la [vista Procesos](../debugger/processes-view.md), use el [cuadro de diálogo Propiedades del proceso](../debugger/process-properties-dialog-box.md).  
  
### Para abrir un cuadro de diálogo Propiedades del proceso  
  
1.  Seleccione un proceso del árbol que aparece en la **vista Procesos**.  
  
2.  En el menú **Ver**, haga clic en **Propiedades**.  
  
 Los cuadros de diálogo de propiedades no son modales, de modo que puede hacer clic en otro elemento de una ventana de vista y el cuadro de diálogo mostrará la información del elemento seleccionado.
