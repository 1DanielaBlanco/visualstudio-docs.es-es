# How to: Display Window Properties
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Para obtener más información sobre una ventana que aparece en la [vista Ventanas](../debugger/windows-view.md), use el [cuadro de diálogo Propiedades de la ventana](../debugger/window-properties-dialog-box.md).  
  
### Para abrir el cuadro de diálogo Propiedades de la ventana  
  
1.  Seleccione el elemento y elija **Propiedades** en el menú **Ver**.  
  
 Los cuadros de diálogo de propiedades no son modales.  Cuando se hace clic en los elementos de una ventana de vista, el cuadro de diálogo se actualiza automáticamente con información sobre cada elemento seleccionado.
