# <a name="how-to-display-window-properties"></a>Cómo: Mostrar propiedades de ventana
Para obtener más información acerca de una ventana que aparece en [Windows Vista](../debugger/windows-view.md), use la [cuadro de diálogo de propiedades de ventana](../debugger/window-properties-dialog-box.md).  
  
### <a name="to-open-the-window-properties-dialog-box"></a>Para abrir el cuadro de diálogo Propiedades de la ventana  
  
1.  Seleccione el elemento y luego elija **propiedades** desde el **vista** menú.  
  
 Cuadros de diálogo de propiedades no son modales. Cuando hace clic en los elementos en una ventana de vista, el cuadro de diálogo se actualiza automáticamente con información sobre cada elemento seleccionado.
