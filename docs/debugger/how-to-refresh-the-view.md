# How to: Refresh the View
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ toma una "instantánea" de las tablas del sistema y actualiza una vista a partir de esta información.  Es importante que se actualicen periódicamente las vistas de sistema.  Si tiene una vista de Spy\+\+ abierta y no la actualiza, no verá los procesos, subprocesos y ventanas que se creen posteriormente.  También es posible que vea elementos que ya no existen.  El comando **Actualizar** está disponible para todas las vistas excepto Mensajes.  
  
### Para actualizar la vista activa  
  
-   Elija **Actualizar** en el menú **Ventana** o haga clic en el botón **Actualizar** situado en la barra de herramientas.
