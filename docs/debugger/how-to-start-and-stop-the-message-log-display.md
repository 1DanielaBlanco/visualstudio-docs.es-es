# How to: Start and Stop the Message Log Display
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Cuando hay una ventana [Vista Mensajes](../debugger/messages-view.md) activa, aparece la opción **Iniciar registro** o **Detener el registro** en el menú **Mensajes** de Spy\+\+ y se activa el botón **Iniciar registro** o **Detener el registro** correspondiente en la barra de herramientas.  
  
### Para iniciar la presentación del registro de mensajes  
  
1.  En el menú **Mensajes**, elija **Iniciar registro**.  
  
### Para detener la presentación del registro de mensajes  
  
1.  En el menú **Mensajes**, elija **Detener el registro**.
