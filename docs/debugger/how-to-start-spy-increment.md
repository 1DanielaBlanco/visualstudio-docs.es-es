# How to: Start Spy++
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Puede iniciar Spy\+\+ desde Visual Studio o en un símbolo del sistema.  
  
 Al iniciar Spy\+\+, si se muestra un mensaje que solicita permiso para realizar cambios en el equipo, haga clic en **Sí**.  
  
> [!NOTE]
>  Solo se puede ejecutar una instancia de Spy\+\+.  Si intenta ejecutar otra instancia, simplemente hace que la instancia que se está ejecutando actualmente obtenga el foco.  
  
### Para iniciar Spy\+\+ desde Visual Studio  
  
-   En el menú **Herramientas**, haga clic en **Spy\+\+**.  
  
     Dado que Spy\+\+ se ejecuta de forma independiente, después de iniciarlo se puede cerrar Visual Studio.  
  
    > [!NOTE]
    >  Cuando se registran mensajes con Spy\+\+, puede hacer que el sistema operativo funcione más lentamente.  
  
### Para iniciar Spy\+\+ en un símbolo del sistema  
  
1.  En una ventana del símbolo del sistema, cambie de directorio a la carpeta que contiene spyxx.exe.  Normalmente, la ruta de acceso de esta carpeta es  \\*carpeta de instalación de Visual Studio*\\Common7\\Tools\\.  
  
2.  Escriba **spyxx.exe** y presione ENTRAR.
