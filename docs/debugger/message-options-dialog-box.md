# Cuadro de di&#225;logo Opciones de mensaje
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use este cuadro de diálogo para seleccionar los mensajes que se muestran en la [vista Mensajes](../debugger/messages-view.md).  Para abrir este cuadro de diálogo, elija **Mensajes de registro** en el menú **Spy**.  
  
## En esta sección  
 [Pestaña Ventanas \(Cuadro de diálogo Opciones de mensaje\)](../debugger/windows-tab-message-options-dialog-box.md)  
 Se usa para seleccionar los tipos de mensajes que se mostrarán.  Incluye la herramienta de búsqueda.  
  
 [Pestaña Mensajes \(Cuadro de diálogo Opciones de mensaje\)](../debugger/messages-tab-message-options-dialog-box.md)  
 Se usa para seleccionar los tipos de mensaje que se van a ver.  
  
 [Pestaña Salida \(Cuadro de diálogo Opciones de mensaje\)](../debugger/output-tab-message-options-dialog-box.md)  
 Se usa para especificar qué datos de mensaje se van a mostrar.
