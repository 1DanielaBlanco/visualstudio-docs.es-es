# Cuadro de di&#225;logo Propiedades del mensaje
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Use este cuadro de diálogo para obtener más detalles de un mensaje concreto.  Para mostrar este cuadro de diálogo, mueva el foco a una ventana de [vista Mensajes](../debugger/messages-view.md).  Seleccione el nodo de un mensaje en el árbol y, a continuación, elija **Propiedades** en el menú **Ver**.  
  
 La pestaña **General** es la única que se muestra.  Están disponibles los siguientes parámetros de configuración:  
  
 **Identificador de ventana**  
 Identificador único de esta ventana.  Los números de identificador de ventana se reutilizan, ya que solo identifican una ventana durante su existencia.  Haga clic en este valor para ver las propiedades de esta ventana.  
  
 **Nivel de anidamiento**  
 Profundidad de anidamiento de este mensaje, donde 0 indica que no hay ningún anidamiento.  
  
 **Mensaje**  
 Número, estado y nombre del mensaje de la ventana que ha seleccionado.  
  
 **lResult**  
 Valor del parámetro *lResult*, si existe.  
  
 **wParam**  
 Valor del parámetro *wParam*, si existe.  
  
 **lParam**  
 Valor del parámetro *lParam*, si existe.  Este valor se descodifica si es un puntero a una cadena o estructura.
