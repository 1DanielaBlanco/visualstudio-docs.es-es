# <a name="run-uwp-apps-from-visual-studio"></a>Ejecutar aplicaciones UWP desde Visual Studio
Aplicaciones UWP se ejecutan en una amplia gama de dispositivos, y Visual Studio ofrece una serie de métodos para ejecutar e interactuar con esas aplicaciones. La experiencia "F5" se ha mejorado para facilitar la depuración, la generación de perfiles y la prueba de aplicaciones durante el desarrollo.
