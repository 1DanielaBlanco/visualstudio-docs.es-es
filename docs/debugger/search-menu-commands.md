# Comandos del men&#250; Buscar
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

El menú **Buscar** proporciona comandos para buscar ventanas concretas.  
  
 **Buscar ventana\/proceso\/subproceso\/mensaje**  
 Abre el cuadro de diálogo **Buscar ventana**, **Buscar proceso**, **Buscar subproceso** o **Buscar mensaje**, donde se puede usar la **herramienta de búsqueda** o bien especificar criterios de búsqueda.  
  
 **Buscar siguiente**  
 Busca la siguiente ventana, proceso, subproceso o mensaje coincidente en la vista actual.  Este elemento de menú \(y el botón relacionado de la barra de herramientas\) sólo estará disponible cuando haya un resultado de búsqueda válido que no sea único.  
  
 **Buscar anterior**  
 Busca la ventana, proceso, subproceso o mensaje anterior coincidente en la vista actual.  Este elemento de menú \(y el botón relacionado de la barra de herramientas\) sólo estará disponible cuando haya un resultado de búsqueda válido que no sea único.
