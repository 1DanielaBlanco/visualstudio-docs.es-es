# Spy++ Views
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Spy\+\+ muestra vistas de árbol de las ventanas, mensajes, procesos y subprocesos disponibles en el equipo.  
  
## En esta sección  
 [Vista Ventanas](../debugger/windows-view.md)  
 Muestra una vista de árbol de las ventanas y controles disponibles.  
  
 [Vista Mensajes](../debugger/messages-view.md)  
 Muestra la secuencia de mensaje asociada a una ventana, proceso o subproceso.  
  
 [Vista Procesos](../debugger/processes-view.md)  
 Muestra una vista de árbol de los procesos activos.  
  
 [Vista de subprocesos](../debugger/threads-view.md)  
 Muestra una vista de árbol de los subprocesos activos.
