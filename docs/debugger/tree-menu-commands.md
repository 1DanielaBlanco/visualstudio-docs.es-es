# Comandos del men&#250; &#193;rbol
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

El menú **Árbol** proporciona comandos para expandir o contraer las vistas.  
  
 **Expandir un nivel**  
 Expande el elemento actualmente seleccionado en una Vista de sistema mostrando un nivel inferior.  
  
 **Expandir bifurcación**  
 Expande todos los niveles situados bajo el elemento actualmente seleccionado.  
  
 **Expandir todo**  
 Expande y muestra todos los niveles en una Vista de sistema.  
  
 **Contraer**  
 Contrae todos los niveles situados bajo el elemento actualmente seleccionado.
