# <a name="debug-session"></a>La sesión de depuración
En cuanto a la arquitectura del depurador, una **sesión**:  
  
-   Es el conjunto de todos los procesos, relacionados o no, que se han iniciado o conectados a desde una sola instancia del IDE.  
  
-   Se implementa mediante un administrador de depuración de sesión (SDM). Para obtener más información, consulte [sesión Debug Manager](../../extensibility/debugger/session-debug-manager.md).
