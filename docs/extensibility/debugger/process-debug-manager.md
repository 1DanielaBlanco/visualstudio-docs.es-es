# Administrador de depuraci&#243;n del proceso
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

El administrador de la depuración \(PDM\) es un componente de Visual Studio que administra los programas y los procesos, y para el administrador de depuración de sesión y los motores de depuración.  
  
 El PDM administra todos los procesos que pueden ser depurados.  Para depurar, un programa se debe registrar con el PDM.  Esto se hace cuando el programa se inicia, por un puerto o un motor de depuración.
