[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Conectado Services SDK de Visual Studio permite a proveedores de servicios de crear extensiones de Visual Studio que los desarrolladores de aplicaciones pueden agregar fácilmente a su proyecto sin dejar el IDE de Visual Studio.
