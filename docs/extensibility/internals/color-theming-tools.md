# <a name="color-theming-tools"></a>Herramientas de creación de temas de color
El SDK de VS proporciona varias herramientas para ayudarle a crear y administrar temas de color.  
  
-   El [Editor de colores de VSIX](../../extensibility/internals/vsix-color-editor.md) herramienta puede crear y editar los colores personalizados para Visual Studio.  
  
-   El [compilador de Color de VSIX](../../extensibility/internals/vsix-color-compiler.md) herramienta convierte archivos de XML de tema de Visual Studio existentes en .pkgdef para la extensión.
