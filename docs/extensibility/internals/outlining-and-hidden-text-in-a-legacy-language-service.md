# Esquematizaci&#243;n y ocultado el texto de un servicio de lenguaje heredado
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

Esquematización permite contraer una secuencia de líneas de texto en una sola línea.  Por ejemplo, en C\+\+ todos los métodos se pueden contraer a una sola línea, mostrando sólo la firma de método.  El texto oculto es la secuencia de líneas de texto que pueden ser mostradas u ocultarse.
