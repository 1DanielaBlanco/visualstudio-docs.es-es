# Elemento MoreInfoURL (esquema del paquete de idioma VSIX)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Opcional. Un vínculo a información localizada sobre la extensión.  
  
## Sintaxis  
  
```  
<MoreInfoURL>URL</MoreInfoURL>  
```  
  
## Atributos y elementos  
 En las siguientes secciones se describen los atributos, los elementos secundarios y los elementos primarios.  
  
### Atributos  
  
|Atributo|Descripción|  
|--------------|-----------------|  
|Ninguna||  
  
### Elementos secundarios  
  
|Elemento|Descripción|  
|--------------|-----------------|  
|Ninguna||  
  
### Elementos primarios  
  
|Elemento|Descripción|  
|--------------|-----------------|  
|[Elemento del paquete de idioma VSIX](../extensibility/vsixlanguagepack-element-vsix-language-pack-schema.md)|Obligatorio. Proporciona el elemento raíz para un paquete de idioma VSIX.|  
  
## Valor de texto  
 Opcional. Un vínculo a un sitio Web. El vínculo es una cadena de texto.

|-|-|

|Puede estar vacío|No es aplicable|
