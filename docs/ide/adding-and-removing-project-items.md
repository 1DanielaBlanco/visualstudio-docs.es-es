# Agregar y quitar elementos del proyecto
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Puede agregar o quitar proyectos y elementos de proyecto haciendo clic con el botón derecho en el nodo de la solución o proyecto en el **Explorador de soluciones** y eligiendo **Agregar** o **Quitar**.  Cuando se quita un proyecto de una solución o un elemento de un proyecto, no se elimina del control de origen ni del disco a menos que se seleccione la opción para eliminarlo de forma permanente.
