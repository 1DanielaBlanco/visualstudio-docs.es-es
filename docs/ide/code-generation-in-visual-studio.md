# <a name="code-generation-in-visual-studio"></a>Generación de código en Visual Studio
Visual Studio puede ayudarle a generar partes de código fuente, lo que le permitirá centrarse en el trabajo real.  Algunas de las características comunes de generación de código incluyen:  

* Generar clases, métodos, tipos, etc.
* Generar comentarios de documentación XML
* Implementar clases abstractas o interfaces
* Introducir una variable local en una expresión compleja

Dado que la lógica de generación de código está estrechamente relacionada con la sintaxis del lenguaje, cada servicio de lenguaje de Visual Studio proporciona sus propias funciones de generación de código. Para obtener más información, consulte las siguientes páginas:  
  
* [Code Generation (C#)](../csharp-ide/code-generation-csharp.md) (Generación de código (C#))
* [Code Generation (Visual Basic)](../vb-ide/code-generation-vb.md) (Generación de código (Visual Basic))

Próximamente estará disponible la documentación para otros lenguajes.
