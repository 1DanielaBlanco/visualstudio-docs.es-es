# <a name="connected-environment"></a>Entorno conectado
Visual Studio es un entorno conectado:  

- Puede iniciar sesión en Visual Studio para habilitar el acceso a todos los servicios asociados a la cuenta de usuario y todas las cuentas vinculadas a ella. La configuración de su cuenta se traspasa allá donde use Visual Studio.  
- Visual Studio le informa cuando cualquier componente de la instalación, como un SDK, tiene una actualización disponible.  
- Puede descargar paquetes y extensiones de NuGet.  
- Y mucho más...
