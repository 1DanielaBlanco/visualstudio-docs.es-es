# <a name="internet-resources-used-by-visual-studio"></a>Recursos de Internet usados por Visual Studio

Visual Studio es un entorno conectado:

- Puede [iniciar sesión](../ide/signing-in-to-visual-studio.md) en Visual Studio para habilitar el acceso a todos los servicios asociados a la cuenta de usuario y todas las cuentas vinculadas a ella. La configuración de su cuenta se traspasa allá donde use Visual Studio.  
- Visual Studio [le notifica](../ide/visual-studio-notifications.md) cuando cualquier componente de la instalación, como un SDK, tiene una actualización disponible.  
- Puede descargar paquetes y extensiones de [NuGet](/nuget/).  
- Puede descargar [extensiones](../ide/finding-and-using-visual-studio-extensions.md) de Visual Studio.  
- Puede conectarse a los servicios de Azure.  
- La página de inicio se actualiza con la novedades más recientes sobre el desarrollador de distintos sitios web.  
- Etc.  

Los temas de esta sesión hacen referencia a las distintas maneras en que Visual Studio usa los recursos de Internet.
