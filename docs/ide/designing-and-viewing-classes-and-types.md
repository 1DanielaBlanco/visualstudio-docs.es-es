# Designing and Viewing Classes and Types
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Diseñe, visualice y refactorice clases y otros tipos del código con el Diseñador de clases de Visual Studio.  Utilice diagramas de clases para crear y editar las clases del proyecto de .NET de Visual C\#, .NET de Visual Basic o C\+\+, comprender mejor la estructura del proyecto o reorganizar el código.  
  
 Aquí tiene más información sobre lo que puede hacer con los diagramas de clases:  
  
-   **Diseñar**: editar el código del proyecto modificando el diagrama de clases.  Agregar nuevos elementos y eliminar los que no quiera.  Los cambios se reflejan en el código.  
  
-   **Visualizar**: comprender la estructura del proyecto viendo las clases del proyecto en un diagrama.  Personalice el diagrama para poder centrarse en los detalles del proyecto que más le importan.  Guarde el diagrama para utilizarlo más adelante en demostraciones o documentación.  
  
-   **Refactorizar**: reemplazar métodos, cambiar el nombre de los identificadores, refactorizar parámetros e implementar interfaces y clases abstractas.
