# Listas de finalizaci&#243;n filtradas en Visual Basic
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

En Visual Basic, las listas de finalización de IntelliSense tienen dos controles de ficha situados cerca de la parte inferior de las listas.  La ficha **Común**, que está seleccionada de manera predeterminada, muestra elementos que se utilizan con mayor frecuencia para finalizar la instrucción que está escribiendo.  La ficha **Todo** muestra todos los elementos que están disponibles para la finalización automática, incluidos aquéllos que también están en la ficha **Común**.
