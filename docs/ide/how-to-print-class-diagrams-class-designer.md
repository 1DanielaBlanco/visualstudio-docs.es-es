# How to: Print Class Diagrams (Class Designer)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Puede imprimir un diagrama de clases con la característica de impresión de Visual Studio.  
  
### Para imprimir un diagrama de clases  
  
1.  Muestre el diagrama de clases.  Si es necesario, seleccione la ficha de diagrama de clases para mostrarlo.  
  
2.  Haga clic en el comando **Imprimir** del menú **Archivo**.  
  
     Se imprimirá el diagrama de clases completo.  Tenga en cuenta que puede ser necesario ajustar los valores del cuadro de diálogo Configurar página para imprimir con un tamaño adecuado.
