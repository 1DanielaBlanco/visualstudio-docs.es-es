# C&#243;mo: Establecer m&#250;ltiples proyectos de inicio
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Visual Studio permite especificar cómo se ejecuta más de un proyecto cuando se inicia el depurador.  
  
### Para establecer varios proyectos de inicio  
  
1.  En el **Explorador de soluciones**, seleccione la solución \(el nodo superior\).  Haga clic con el botón derecho en el nodo para abrir el menú contextual.  
  
2.  Seleccione **Propiedades**.  Se abrirá el cuadro de diálogo **Páginas de propiedades de la solución**.  
  
3.  Expanda el nodo **Propiedades comunes** y haga clic en **Proyecto de inicio**.  
  
4.  Haga clic en **Proyectos de inicio múltiples** y establezca las acciones apropiadas.  Para obtener más información, vea [Cómo: Modificar las propiedades y los valores de configuración del proyecto](http://msdn.microsoft.com/es-es/e7184bc5-2f2b-4b4f-aa9a-3ecfcbc48b67).
