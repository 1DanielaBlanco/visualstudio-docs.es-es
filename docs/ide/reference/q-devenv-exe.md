# <a name="-devenvexe"></a>/? (devenv.exe)
Muestra un cuadro de mensaje que enumera todos los modificadores `devenv`, con una breve descripción de cada uno de ellos.

## <a name="syntax"></a>Sintaxis

```
devenv /?
```
