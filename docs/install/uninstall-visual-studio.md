# <a name="uninstall-visual-studio"></a>Desinstalar Visual Studio
Esta página le guía a través de la desinstalación de Visual Studio, nuestro conjunto integrado de herramientas de productividad para desarrolladores.  

1.  Busque el instalador de Visual Studio en su equipo.  

     Por ejemplo, en un equipo que ejecuta la Actualización de aniversario de Windows 10, haga clic en **Iniciar** y, después, desplácese hasta la letra **I** donde lo verá como **Instalador de Microsoft Visual Studio**.  

     ![00-ModifyingDev15Prev5-FindTheVisualStudioInstaller](../install/media/00-modifyingdev15prev5-findthevisualstudioinstaller.png)

2.  Haga clic para iniciar el instalador, haga clic en el icono ![Icono de detalles](media/vs2017uninstall-UninstallIcon.png) y, después, haga clic en **Desinstalar**.  

     ![Modificar Visual Studio 2017 RC; Iniciar o modificar](media/vs2017uninstall-ChooseUninstallFromInstaller.png "Reparar o desinstalar Visual Studio 2017 RC")  

3.  Haga clic en **Aceptar** para confirmar su elección.

     ![Modificar Visual Studio 2017 RC; Iniciar o modificar](media/vs2017uninstall-UninstallConfirm.png "Confirmar la desinstalación de Visual Studio 2017 RC")  

Si cambia de idea más tarde y quiere volver a instalar Visual Studio 2017 RC, simplemente inicie el instalador de Visual Studio de nuevo y, después, haga clic en **Instalar** desde la pantalla de selección.

> [!IMPORTANT]
> A pesar de que, en general, se admite el uso de Visual Studio 2017 RC en un entorno de producción, las cargas de trabajo y los componentes que aparecen marcados como "Versión preliminar" en la interfaz de usuario de instalación no son compatibles para usarlos en un entorno de producción.
