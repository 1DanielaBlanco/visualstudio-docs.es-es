# &#39;&lt;nombreDeEnsamblado&#39; es un ensamblado y no se puede hacer referencia a &#233;l como m&#243;dulo.
Se intentó usar un ensamblado como un módulo.  
  
 **Identificador de error:** BC31077  
  
### Para corregir este error  
  
-   Cambie la referencia a un módulo.
