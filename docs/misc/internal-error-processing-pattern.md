# Internal error processing pattern.
Este error suele producirse cuando se ha incluido una expresión regular en una cadena de búsqueda pero la sintaxis de la expresión regular contiene errores.  
  
### Para corregir este error  
  
1.  Vuelva a especificar la cadena de búsqueda utilizando la sintaxis de expresión regular adecuada.  Para obtener más información, vea [Expresiones regulares](../ide/using-regular-expressions-in-visual-studio.md).
