# Error de MSBuild MSB1039
**MSB1039: Especifique la versión del conjunto de herramientas.**  
  
 No se especificó ningún valor cuando se utilizó el modificador **\/ToolsVersion**.  
  
### Para corregir este error  
  
-   Cuando utilice el modificador **\/ToolsVersion**, asegúrese de especificar un valor válido no vacío para `ToolsVersion`.
