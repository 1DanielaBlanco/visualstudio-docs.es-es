# Error de MSBuild MSB3171
**MSB3171: Existe un problema al generar el manifiesto. '  \<error\>'**  
  
 Se trata de un error inesperado que se genera al obtener información sobre el archivo o ensamblado al que se hace referencia.  Este mensaje puede generarse durante la generación del manifiesto de aplicación o de implementación.  
  
### Para corregir este error  
  
-   Para obtener más información sobre la causa de un error de MSBuild y las posibles soluciones, vea [Recursos adicionales](../msbuild/additional-msbuild-resources.md).
