# Several Documents About Server Explorer Are Available
Ayuda de[!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] contiene los siguientes documentos sobre **Explorador de servidores** que puede aplicar a su solicitud de Ayuda. Seleccione el documento que corresponda mejor a la tarea que está intentando realizar.  
  
|Título|Descripción|  
|------------|-----------------|  
|[Add Server Dialog Box](../Topic/Add%20Server%20Dialog%20Box.md)|Describe el cuadro de diálogo **Agregar servidor**, que puede utilizar para especificar que se muestre un servidor en el **Explorador de servidores**.|  
|[Recursos adicionales para la solución de problemas de errores de acceso a datos](../data-tools/additional-resources-for-troubleshooting-data-access-errors.md)|Vínculos a documentos sobre servicios de datos.|  
|[Connect As Dialog Box](../Topic/Connect%20As%20Dialog%20Box.md)|Describe el cuadro de diálogo **Conectar como**, que permite cambiar la seguridad de inicio de sesión de un servidor especificado.|  
|[Create Message Queue Dialog Box](../Topic/Create%20Message%20Queue%20Dialog%20Box.md)|Describe el cuadro de diálogo **Crear cola de mensajes**, que permite crear y poner nombre a una cola en el servidor especificado.|  
|[Server Explorer](../Topic/Server%20Explorer.md)|Describe el **Explorador de servidores**\/**Explorador de bases de datos** que es la consola de administración de servidores de [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].|  
|[Examinar las conexiones de SharePoint utilizando el Explorador de servidores](/office-dev/office-dev/browsing-sharepoint-connections-using-server-explorer)|Describe cómo examinar los sitios de SharePoint mediante **Explorador de servidores**.|  
|[Información general de datos locales](../data-tools/local-data-overview.md)|Describe cómo convertir un archivo de base de datos \(.mdf\) en un formato utilizado por [!INCLUDE[vs_dev11_long](../data-tools/includes/vs_dev11_long_md.md)].|
