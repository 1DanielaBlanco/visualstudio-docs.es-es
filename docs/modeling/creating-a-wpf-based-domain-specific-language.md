 Información y ejemplos acerca de esta característica pueden encontrarse en el sitio Web de Visual Studio de visualización y modelado de herramientas en [http://go.microsoft.com/fwlink/?LinkId=186128](http://go.microsoft.com/fwlink/?LinkId=186128)
