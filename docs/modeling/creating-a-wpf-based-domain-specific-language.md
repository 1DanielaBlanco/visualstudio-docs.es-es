# <a name="create-a-wpf-based-domain-specific-language"></a>Crear un lenguaje específico de dominio basado en WPF

Puede crear un lenguaje específico de dominio que tiene un diseñador WPF en lugar de un diseñador gráfico.

Información y ejemplos acerca de esta característica pueden encontrarse en el Visual Studio de visualización y el sitio Web de herramientas de modelado en [http://go.microsoft.com/fwlink/?LinkId=186128](http://go.microsoft.com/fwlink/?LinkId=186128)
