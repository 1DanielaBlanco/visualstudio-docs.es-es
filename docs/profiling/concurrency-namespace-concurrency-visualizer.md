# Espacio de nombres de simultaneidad (Visualizador de simultaneidad)
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

El espacio de nombres `Concurrency` proporciona las clases y funciones que dan acceso al Runtime de simultaneidad, un marco de programación simultáneo para C\+\+.  Para obtener más información, vea [Runtime de simultaneidad](/visual-cpp/parallel/concrt/concurrency-runtime).  
  
## Sintaxis  
  
```  
namespace Concurrency;  
```  
  
## Miembros  
  
### Espacios de nombres  
  
|Name|Descripción|  
|----------|-----------------|  
|[diagnostic \(Espacio de nombres\)](../profiling/diagnostic-namespace.md)|El espacio de nombres `diagnostics` proporciona la funcionalidad para emitir los marcadores del Visualizador de simultaneidad.|  
  
## Requisitos  
 **Encabezado:** cvmarkersobj.h
