# C&#243;mo: Ver advertencias de rendimiento de herramientas de generaci&#243;n de perfiles
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Las advertencias de rendimiento de las herramientas de generación de perfiles de [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)] indican problemas en una aplicación para la que se ha generado perfiles que puede ralentizar la ejecución de programas.  Las advertencias también pueden indicar que debe cambiar los métodos de recolección para recopilar datos más útiles.  Las advertencias de rendimiento se generan automáticamente en una sesión de generación de perfiles y aparecen en la ventana **Lista de errores** cuando se abre un archivo de datos de generación de perfiles en el IDE de [!INCLUDE[vsprvs](../code-quality/includes/vsprvs_md.md)].  
  
 Si no aparece la ventana **Lista de errores** al abrir un archivo de datos de generación de perfiles, haga clic en el menú **Ver** y, a continuación, haga clic en **Lista de errores**.
