# Hay m&#250;ltiples categor&#237;as
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Se selecciona una sección que contiene más de un tipo de actividad para un intervalo de tiempo determinado.  Observe que se muestran dos colores, uno apilado verticalmente sobre el otro.  Estos colores representan simplemente las dos actividades más frecuentes en el intervalo de tiempo determinado.  Las frecuencias de estas dos actividades se muestran relativas entre sí como tercios o mitades de la barra vertical.  Ver más detalles, pruebe a hacer zoom en esta sección de la vista de subproceso.
