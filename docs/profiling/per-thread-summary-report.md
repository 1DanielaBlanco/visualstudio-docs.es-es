# Informe de resumen por subproceso
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

Este gráfico de barras muestra la proporción de tiempo que cada subproceso pasa en cada categoría de actividad durante el intervalo de tiempo visible. La “ejecución” significa que el subproceso se está ejecutando; el resto de las categorías indican que el subproceso está esperando algo.
