# Vistas de interacci&#243;n de capas
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

La generación de perfiles de interacción de capas proporciona información adicional sobre los tiempos de ejecución de funciones de aplicaciones de varias capas que se comunican con bases de datos a través de [!INCLUDE[vstecado](../data-tools/includes/vstecado_md.md)].  Los datos se recopilan solamente para las llamadas a funciones sincrónicas.  Puede incluir datos de interacción de capas en todos los métodos de generación de perfiles.
