# <a name="rtvs-current-version"></a>Versión actual de RTVS

La última versión de Herramientas de R para Visual Studio (RTVS) es **1.0.30321.1407** (recomendada).

Se trata de la versión de lanzamiento 1.0 de RTVS, que está disponible en [aka.ms/rtvs-current](https://aka.ms/rtvs-current).

Para comprobar la versión, vaya a **Ayuda > Acerca de** en Visual Studio y desplácese hacia abajo hasta la entrada **Herramientas de R para Visual Studio**:

![Examinar la versión de RTVS en Visual Studio (VS2017)](~/rtvs/media/current-version.png)

Consulte los requisitos de [instalación](installation.md) de Visual Studio y Windows.

Para obtener actualizaciones de Microsoft R Open y Microsoft R Server, consulte: [Productos de Microsoft R](http://aka.ms/rtvs-msft-r).
