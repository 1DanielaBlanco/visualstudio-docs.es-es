## <a name="resources"></a>Recursos

- [Documentación](index.md)
- [Pasos de instalación](installation.md)
- [Enviar y examinar errores y solicitudes de características en GitHub](https://github.com/Microsoft/RTVS/issues)

## <a name="microsoft-r-distributions"></a>Distribuciones de Microsoft R

RTVS admite CRAN R y distribuciones de Microsoft R, incluida la versión de R distribuida con SQL Server 2016. Vea los vínculos siguientes para obtener más información sobre las distintas distribuciones de Microsoft R:

* [Microsoft R Open](https://mran.microsoft.com/download/)
* [Microsoft R Client](https://msdn.microsoft.com/microsoft-r/r-client-get-started)
* [Microsoft R Server](https://www.microsoft.com/server-cloud/products/r-server/)
