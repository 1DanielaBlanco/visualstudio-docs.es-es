# C&#243;mo: Agregar una caracter&#237;stica a las soluciones de SharePoint
  Al crear una solución de SharePoint, se agrega una característica y un paquete.  Puede agregar más elementos de característica para organizar tipos diferentes de archivos de SharePoint.  
  
### Para agregar una característica mediante el Explorador de soluciones  
  
1.  En **Explorador de soluciones**, abra el menú contextual para **Características**y, a continuación **Agregue la característica**.
