# <a name="sharepoint-development-samples"></a>Ejemplos de desarrollo de SharePoint
  Puede usar Visual Studio para descargar e instalar ejemplos de aplicaciones de SharePoint desde el [MSDN Code Gallery](http://go.microsoft.com/fwlink/?LinkId=254185).  
  
 Puede descargar cada ejemplo individualmente o puede descargar un paquete de muestra, que contiene ejemplos relacionados que comparten una tecnología o un tema. Recibirá una notificación cuando se publiquen cambios en el código fuente de cualquier ejemplo que descargue.
