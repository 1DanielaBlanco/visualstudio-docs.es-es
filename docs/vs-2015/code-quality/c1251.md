La versión más reciente de este tema puede encontrarse en [C1251](https://docs.microsoft.com/visualstudio/code-quality/c1251).  
  
Advertencia C1251: no se puede cargar los modelos.  
  
 La herramienta de análisis de código informa de esta advertencia cuando hay un error interno en el archivo de modelo, no en el código que se está analizando.
