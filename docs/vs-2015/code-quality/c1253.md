# <a name="c1253"></a>C1253
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [C1253](https://docs.microsoft.com/visualstudio/code-quality/c1253).  
  
Advertencia C1253: no se puede cargar el archivo de modelo.  
  
 La herramienta de análisis de código informa de esta advertencia cuando hay un error interno en el archivo de modelo, no en el código que se está analizando.
