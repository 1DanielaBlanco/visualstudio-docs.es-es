# <a name="c1254"></a>C1254
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [C1254](https://docs.microsoft.com/visualstudio/code-quality/c1254).  
  
Advertencia C1254: discrepancia de versiones del complemento: versión no coincide con la versión del controlador de PREfast  
  
 La herramienta de análisis de código informa de esta advertencia cuando hay un error interno con la versión del complemento, no en el código que se está analizando.
