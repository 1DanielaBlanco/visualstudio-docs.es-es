# <a name="c28216"></a>C28216
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [C28216](https://docs.microsoft.com/visualstudio/code-quality/c28216).  
  
Advertencia C28216: la _Check_return\_ anotación solo se aplica a las condiciones posteriores para el parámetro de función específica.  
  
 El `_Check_return_` anotación se ha aplicado en un contexto distinto `__post`; puede que necesite un `__post` (o `__drv_out`) modificador, o bien es posible que no esté colocado correctamente.
