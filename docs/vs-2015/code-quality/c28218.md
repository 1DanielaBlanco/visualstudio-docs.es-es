# <a name="c28218"></a>C28218
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [C28218](https://docs.microsoft.com/visualstudio/code-quality/c28218).  
  
Advertencia C28218: para el parámetro de función, el parámetro de la anotación no coincide con el encontrado en el archivo  
  
 Las anotaciones en la línea actual y en la línea en el mensaje son incoherentes. Esto no debería ser posible si se utilizan las macros estándares para anotaciones. Esta advertencia no se espera que se muestra en el uso normal.
