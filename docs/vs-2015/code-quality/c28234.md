# <a name="c28234"></a>C28234
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28234: _At\_ expresión no se aplica a la función actual  
  
 Esta advertencia indica que el valor de un `_At_` expresión no identifica un objeto accesible.
