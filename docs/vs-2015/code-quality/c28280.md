# <a name="c28280"></a>C28280
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [C28280](https://docs.microsoft.com/visualstudio/code-quality/c28280).  
  
Advertencia C28280: para el símbolo, se encontró un 'end' sin el correspondiente 'begin'  
  
 El lenguaje de anotación es compatible con un inicio y fin (`{` y `}` en C) ha perdido desequilibrada la construcción y el emparejamiento. Esta situación puede evitarse si se utilizan las macros.
