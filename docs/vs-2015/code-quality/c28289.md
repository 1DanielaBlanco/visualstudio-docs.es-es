# <a name="c28289"></a>C28289
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28289: para la función: ReadableTo o WritableTo no tenían una especificación de límite como parámetro  
  
 La herramienta de análisis de código emite esta advertencia cuando se miscoded la anotación del parámetro de función y como se indicó.
