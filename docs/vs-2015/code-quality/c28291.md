# <a name="c28291"></a>C28291
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Advertencia C28291: null Post/NotNull posterior en el nivel 0 carece de sentido para la función \<x > en param \<número >  
  
 La herramienta de análisis de código informa de esta advertencia cuando se especifica la condición posterior de un parámetro de nivel cero desreferencia una propiedad es null y no null. Este error se produce porque se desreferencia un valor en el nivel cero no se puede cambiar.
