# <a name="idiasymbolgetisdataaligned"></a>IDiaSymbol::get_isDataAligned
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [Get_isdataaligned](https://docs.microsoft.com/visualstudio/debugger/debug-interface-access/idiasymbol-get-isdataaligned).  
  
Recupera una marca que especifica si el tipo definido por el usuario (UDT) se ha alineado a algunos límites de memoria concreta.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp  
HRESULT get_isDataAligned(  
   BOOL *pFlag  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pFlag`  
 [out] Devuelve `TRUE` si el UDT se ha alineado a algunos límites de memoria; en caso contrario, devuelve `FALSE`.  
  
## <a name="return-value"></a>Valor devuelto  
 Si es correcto, devuelve `S_OK`; en caso contrario, devuelve `S_FALSE` o código de error.  
  
> [!NOTE]
>  Un valor devuelto de `S_FALSE` significa que la propiedad no está disponible para el símbolo.  
  
## <a name="remarks"></a>Comentarios  
 Esta propiedad se establece generalmente cuando se compila el archivo ejecutable con la alineación de datos no predeterminados. Por ejemplo, el compilador de C++ de Microsoft puede cambiar la alineación de datos con la opción de línea de comandos, /Zp*#*, donde *#* es un valor de byte.
