# <a name="error-mixed-mode-debugging-for-ia64-processes-is-unsupported"></a>Error: No se admite la depuración en modo mixto para procesos IA64
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [Error: no se admite la depuración en modo mixto para los procesos IA64](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-for-ia64-processes-is-unsupported).  
  
El depurador [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] no permite depurar código nativo y administrado mixto en un proceso basado en Itanium.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compile una versión de 32 bits de su aplicación para la depuración.
