# <a name="error-mixed-mode-debugging-is-supported-only-when-using-microsoft-net-framework-20-or-greater"></a>Error: El modo de depuración mixto se admite solo cuando se usa Microsoft .NET Framework 2.0 o superior
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [Error: la depuración en modo mixto se admite solo cuando se usa Microsoft .NET Framework 2.0 o superior](https://docs.microsoft.com/visualstudio/debugger/error-mixed-mode-debugging-is-supported-only-when-using-microsoft-dotnet-framework-2-0-or-greater).  
  
Para depurar código nativo y administrado, debe tener la versión 2.0, 3.0 de [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)]. 3.5 ó 4. Depuración en modo mixto con versiones anteriores de la [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] no se admite.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Actualice [!INCLUDE[dnprdnshort](../includes/dnprdnshort-md.md)] a la versión 2.0, 3.0, 3.5 o 4.
