# <a name="how-to-display-spy-views"></a>Cómo: Mostrar las vistas de Spy++
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [Cómo: mostrar vistas de Spy ++](https://docs.microsoft.com/visualstudio/debugger/how-to-display-spy-increment-views).  
  
Cuando se inicia Spy ++, se abre en [Windows Vista](../debugger/windows-view.md), vista de árbol de todas las ventanas y controles disponibles en el equipo. Esta ventana raíz es títulos "Windows 1". También hay tres otras vistas de Spy ++: [vista mensajes](../debugger/messages-view.md), [vista procesos](../debugger/processes-view.md), y [vista de subprocesos](../debugger/threads-view.md). Puede mostrar la vista deseada desde la **Spy** menú. Para obtener más información, consulte [vistas de Spy ++](../debugger/spy-increment-views.md).
