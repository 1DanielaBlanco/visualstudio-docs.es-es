# <a name="tree-menu-commands"></a>Comandos del menú Árbol
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [comandos del menú árbol](https://docs.microsoft.com/visualstudio/debugger/tree-menu-commands).  
  
El **árbol** menú proporciona comandos para expandir o contraer las vistas.  
  
 **Expandir un nivel**  
 Expande el elemento seleccionado actualmente en una vista del sistema mostrando un nivel por debajo de él.  
  
 **Expandir rama**  
 Expande todos los niveles por debajo del elemento seleccionado actualmente.  
  
 **Expandir todo**  
 Se expande y muestra todos los niveles en una vista del sistema.  
  
 **Contraer**  
 Contrae todos los niveles por debajo del elemento seleccionado actualmente.
