# <a name="asyncvoidmethodbuilderobjectidfordebugger-property"></a>AsyncVoidMethodBuilder.ObjectIdForDebugger (Propiedad)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [AsyncVoidMethodBuilder.ObjectIdForDebugger (propiedad)](https://docs.microsoft.com/visualstudio/extensibility/debugger/asyncvoidmethodbuilder-objectidfordebugger-property).  
  
Obtiene un objeto que puede utilizarse para identificar de forma única este generador para el depurador.  
  
 **Namespace:** <xref:System.Runtime.CompilerServices?displayProperty=fullName>  
  
 **Ensamblado:** mscorlib (en mscorlib.dll)  
  
 Dado que no se puede obtener acceso a este miembro interno de .NET Framework, la sintaxis siguiente se proporciona el lenguaje intermedio en común (CIL).  
  
## <a name="syntax"></a>Sintaxis  
  
```csharp  
private object ObjectIdForDebugger  
```
