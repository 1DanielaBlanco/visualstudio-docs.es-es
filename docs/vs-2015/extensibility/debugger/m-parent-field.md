# <a name="mparent-field"></a>m_parent (Campo)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [m_Parent (campo)](https://docs.microsoft.com/visualstudio/extensibility/debugger/m-parent-field).  
  
El campo de respaldo para el <xref:System.Threading.Tasks.Task?displayProperty=fullName> propiedad principal.  
  
 **Namespace:** <xref:System.Threading.Tasks?displayProperty=fullName>  
  
 **Ensamblado:** mscorlib (en mscorlib.dll)  
  
 Dado que no se puede obtener acceso a este miembro interno de .NET Framework, la sintaxis siguiente se proporciona el lenguaje intermedio en común (CIL).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
.field private initonly class System.Threading.Tasks.Task m_parent  
```
