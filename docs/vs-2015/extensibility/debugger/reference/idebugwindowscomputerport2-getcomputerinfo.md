# <a name="idebugwindowscomputerport2getcomputerinfo"></a>IDebugWindowsComputerPort2::GetComputerInfo
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [IDebugWindowsComputerPort2::GetComputerInfo](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/idebugwindowscomputerport2-getcomputerinfo).  
  
Recupera información sobre el equipo en el que el depurador en ejecución.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp#  
HRESULT GetComputerInfo(  
   COMPUTER_INFO * pInfo  
);  
```  
  
```csharp  
public int GetComputerInfo(  
   out COMPUTER_INFO[] pInfo  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pInfo`  
 [out] Referencia a una estructura que contiene la información del equipo.  
  
## <a name="return-value"></a>Valor devuelto  
 Si es correcto, devuelve `S_OK`; en caso contrario, devuelve un código de error.
