# <a name="namematch"></a>NAME_MATCH
[!INCLUDE[vs2017banner](../../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [NAME_MATCH](https://docs.microsoft.com/visualstudio/extensibility/debugger/reference/name-match).  
  
Selecciona la opción mayúsculas para la coincidencia de nombres.  
  
## <a name="syntax"></a>Sintaxis  
  
```cpp#  
typedef enum {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
} NAME_MATCH;  
```  
  
```csharp  
public enum NameMatchOptions {   
   nmNone            = 0,  
   nmCaseSensitive   = 1,  
   nmCaseInsensitive = 2  
}  
```  
  
## <a name="members"></a>Miembros  
 nmNone  
 No se especifican opciones.  
  
 nmCaseSensitive  
 Indica que los nombres deben coincidir distinguen mayúsculas de minúsculas.  
  
 nmCaseInsensitive  
 Indica que deben coincidir los nombres no distinguen mayúsculas de minúsculas.
