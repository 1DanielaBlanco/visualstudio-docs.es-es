# <a name="set-current-stack-frame-command"></a>Establecer marco de pila actual (Comando)
[!INCLUDE[vs2017banner](../../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [establecer comando de marco de pila actual](https://docs.microsoft.com/visualstudio/ide/reference/set-current-stack-frame-command).  
  
  
Le permite establecer un marco de pila determinado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
Debug.SetCurrentStackFrame index  
```  
  
## <a name="arguments"></a>Argumentos  
 `index`  
 Requerido. Selecciona un marco de pila por su índice.  
  
## <a name="example"></a>Ejemplo  
  
```  
>Debug.SetCurrentStackFrame 1  
```
