# <a name="vspackage-essentials"></a>Elementos fundamentales de VSPackage
Los VSPackages son módulos de software que conforman y amplían Visual Studio con elementos de interfaz de usuario, servicios, proyectos, editores y diseñadores. Los VSPackages son la unidad central de la arquitectura de [!INCLUDE[vsprvs](../includes/vsprvs-md.md)]. [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] en sí mismo está escrito principalmente como una colección de VSPackages.  
  
 Para crear un [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] aplicación de extensibilidad, primero cree un VSPackage. A continuación, puede agregar comandos de menú, ventanas de herramientas, ventanas de documento u otro código para extender el [!INCLUDE[vsprvs](../includes/vsprvs-md.md)] IDE.  
  
 Para usar VSPackages, debe instalar el SDK de Visual Studio. Para seguir este tutorial, debe instalar el SDK de Visual Studio. Para obtener más información, consulte [SDK de Visual Studio](../extensibility/visual-studio-sdk.md).
