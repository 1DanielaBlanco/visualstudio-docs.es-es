# <a name="spanspan-destructor"></a>span::~span (Destructor)
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

La versión más reciente de este tema puede encontrarse en [span:: ~ span (destructor)](https://docs.microsoft.com/visualstudio/profiling/span-tilde-span-destructor).  
  
Destruye el objeto de `span` y libera sus recursos.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
~span();  
```  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** cvmarkersobj.h  
  
 **Espacio de nombres:** Concurrency::diagnostic
