# Microsoft Office no est&#225; instalado en este equipo
  Visual Studio no encuentra la aplicación de Microsoft Office que se corresponde al tipo de proyecto. Si está desarrollando un proyecto de complemento de VSTO, significa que no puede depurar o ejecutar el proyecto. Si intenta crear un proyecto de nivel de documento, significa que no puede crear el proyecto.  
  
### Para corregir este error  
  
1.  Instale la aplicación Microsoft Office asociada a este tipo de proyecto, incluidos los ensamblados de interoperabilidad principales. Para obtener más información sobre las versiones compatibles de Microsoft Office, vea [Configurar un equipo para desarrollar soluciones de Office](../vsto/configuring-a-computer-to-develop-office-solutions.md).
