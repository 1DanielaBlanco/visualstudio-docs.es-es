# Trabajar con documentos de Visio
  En los temas de esta sección se proporcionan procedimientos paso a paso y ejemplos de código para usar el modelo de objetos de Microsoft Office Visio para trabajar con documentos de proyectos de Office.  
  
|Tarea|Procedimiento|  
|-----------|-------------------|  
|Cree un documento de Visio en blanco o un nuevo documento de Visio basado en una plantilla, una galería de símbolos o un documento existente.|[Cómo: Crear nuevos documentos de Visio mediante programación](../vsto/how-to-programmatically-create-new-visio-documents.md)|  
|Abra un documento de Visio existente y especifique las opciones de lectura y escritura, solo lectura y acoplamiento.|[Cómo: Abrir documentos de Visio mediante programación](../vsto/how-to-programmatically-open-visio-documents.md)|  
|Cierre el documento de Visio activo.|[Cómo: Cerrar documentos de Visio mediante programación](../vsto/how-to-programmatically-close-visio-documents.md)|  
|Guardar un documento con el mismo nombre o con un nombre nuevo.|[Cómo: Guardar documentos de Visio mediante programación](../vsto/how-to-programmatically-save-visio-documents.md)|  
|Imprima un documento completo o una página concreta.|[Cómo: Imprimir documentos de Visio mediante programación](../vsto/how-to-programmatically-print-visio-documents.md)|
