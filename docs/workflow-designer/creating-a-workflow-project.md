# <a name="creating-a-workflow-project"></a>Crear un proyecto de flujo de trabajo

Los flujos de trabajo, servicios de flujo de trabajo de Windows Communication Foundation (WCF), las actividades personalizadas y diseñadores de actividad personalizados se pueden crear mediante plantillas de proyecto de Visual Studio. En esta sección se describe cómo crear bibliotecas y aplicaciones con las plantillas de proyecto disponibles en Visual Studio.

Los primeros cuatro temas siguientes describen cómo utilizar las plantillas de flujo de trabajo se encuentra en la **nuevo proyecto** ventana. Pueden tener acceso a estas plantillas abriendo Visual Studio 2010 y seleccionando **archivo** > **New** > **proyecto**, la  **Flujo de trabajo** categoría desde la **plantillas instaladas** sección de la izquierda, bajo la **Visual Basic** o **Visual C#** nodo.

El último tema describe cómo agregar actividades de flujo de trabajo, diseñadores y servicios u otro proyecto de Visual Studio elementos a un flujo de trabajo existente (u otro).
