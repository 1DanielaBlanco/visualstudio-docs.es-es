# <a name="debugging-workflows-with-the-workflow-designer"></a>Depurar flujos de trabajo con el Diseñador de flujo de trabajo

El Diseñador de flujo de trabajo de Windows proporciona la capacidad para depurar flujos de trabajo y actividades personalizadas. El proceso y comportamiento son similares a los del depurador [!INCLUDE[vs2010](../misc/includes/vs2010_md.md)] predeterminado. Los temas en esta sección describen cómo efectuar procedimientos de depuración tradicionales así como depurar código XAML y expresiones.
