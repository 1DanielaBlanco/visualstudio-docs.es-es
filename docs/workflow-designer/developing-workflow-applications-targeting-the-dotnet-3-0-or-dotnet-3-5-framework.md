# <a name="developing-workflow-applications-targeting-the-net-30-or-net-35-framework"></a>Desarrollar aplicaciones de flujo de trabajo que tienen como destino .NET 3.0 o .NET 3.5 Framework

Visual Studio 2010 proporciona el Diseñador de flujo de trabajo de Windows para crear aplicaciones de Windows Workflow Foundation (WF) que tienen como destino .NET Framework 4. El diseñador utiliza la familiar interfaz de usuario de Visual Studio para la creación gráfica y depuración de aplicaciones de flujo de trabajo.

Visual Studio 2010 también proporciona acceso a un diseñador de flujo de trabajo heredado que puede utilizarse como destino la versión 3.5 de .NET Framework o el conocido como WinFX. Esta sección contiene indicaciones sobre el uso de este diseñador heredado.
