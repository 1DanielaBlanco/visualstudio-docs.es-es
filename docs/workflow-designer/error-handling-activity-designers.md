# <a name="error-handling-activity-designers"></a>Diseñadores de actividades de control de errores

Los diseñadores de actividades de control de errores se usan para administrar el control de errores en el Diseñador de flujo de trabajo de Windows. Los temas contenidos en esta sección proporcionan instrucciones sobre cómo usar las actividades del Diseñador de flujo de trabajo y los diseñadores de plantilla.
