# C&#243;mo: Configurar las propiedades de proyectos de flujo de trabajo (Heredado)
Siga estos pasos para ver o modificar las propiedades asociadas a proyectos de flujo de trabajo.  
  
### Para ver o modificar las propiedades de los proyectos de flujo de trabajo  
  
1.  Si la ventana **Explorador de soluciones** no está visible, seleccione **Explorador de soluciones** en el menú **Ver**.  
  
2.  En el **Explorador de soluciones**, seleccione un proyecto.  
  
3.  Si la ventana **Propiedades** no está visible, seleccione **Ventana Propiedades** en el menú **Ver**.  
  
4.  En la ventana **Propiedades** del proyecto, vea y modifique las propiedades que desee.
