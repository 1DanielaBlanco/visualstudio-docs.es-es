# <a name="state-machine-activity-designers"></a>Diseñadores de actividad de equipo de estado
Diseñador de flujo de trabajo de Windows incluye un número de diseñadores de actividad que se pueden utilizar cuando cree flujos de trabajo. Esta sección contiene los diseñadores de actividad proporcionados por el sistema que se usan para crear y administrar flujos de trabajo de máquina de estados. Los siguientes temas describen estos diseñadores y ofrecen orientación sobre cómo usarlas.

## <a name="in-this-section"></a>En esta sección
 [StateMachine](../workflow-designer/statemachine-activity-designer.md) contiene una colección de Estados y modelos de flujos de trabajo mediante el paradigma de la máquina de estado conocido.

 [Estado](../workflow-designer/state-activity-designer.md) representa un estado en el que puede estar una máquina de Estados.

 [FinalState](../workflow-designer/finalstate-activity-designer.md) representa un estado que finaliza una instancia de la máquina de Estados.

 [Transición](../workflow-designer/transition-activity-designer.md) representa la transición entre dos Estados.
