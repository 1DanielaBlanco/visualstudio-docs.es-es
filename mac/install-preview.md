# <a name="installing-preview-releases"></a>Instalación de versiones preliminares

Antes de que se lance oficialmente una nueva versión de Visual Studio para Mac, se lanza como versión preliminar. La versión preliminar le da la oportunidad de probar las nuevas características y obtener las últimas correcciones de errores antes de que se incorporen completamente al producto.

Las versiones preliminares de Visual Studio para Mac se distribuyen como actualizaciones, en lugar de mediante una descarga independiente. Visual Studio para Mac tiene tres canales de actualización, como se describe en el artículo [update](~/update.md) (actualización): Stable, Beta, and Alpha.

La mayoría de las versiones preliminares estarán disponibles a través de los canales **Beta** y **Alpha**, pero compruebe siempre las [notas de la versión preliminar](/visualstudio/releasenotes/vs2017-mac-preview-relnotes) para ver información más precisa.

Para instalar la versión preliminar de Visual Studio para Mac, siga estos pasos:

1. Vaya a **Visual Studio > Buscar actualizaciones...**.
2. En el cuadro combinado del canal de actualización, seleccione **Beta**.
3. Seleccione el botón **Switch channel** (Conmutador de canal) para cambiar al canal seleccionado e iniciar la descarga de nuevas actualizaciones.
4. Seleccione el botón **Reiniciar e instalar actualizaciones** para iniciar la instalación de actualizaciones.

Para obtener más información sobre las actualizaciones en Visual Studio para Mac, consulte el artículo [Update](~/update.md) (Actualización).
