# <a name="jspropertyidref-typedef"></a>JsPropertyIdRef (Typedef)
Identificador de propiedad.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
typedef JsRef JsPropertyIdRef;  
```  
  
## <a name="remarks"></a>Comentarios  
 Los identificadores de propiedad se utilizan para hacer referencia a propiedades de objetos JavaScript en lugar de usar cadenas.  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** jsrt.h
