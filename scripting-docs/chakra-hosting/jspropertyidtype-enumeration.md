# <a name="jspropertyidtype-enumeration"></a>Enumeración JsPropertyIdType
Enumeración de tipo de una propiedad de JavaScript.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
enum JsPropertyIdType;  
```  
  
## <a name="members"></a>Miembros  
  
### <a name="values"></a>Valores  
  
|Nombre|Descripción|  
|----------|-----------------|  
|JsPropertyIdTypeString|Enumeración de tipo de una propiedad de cadena de JavaScript.|  
|JsPropertyIdTypeSymbol|Enumeración de tipo de una propiedad de símbolo de JavaScript.|  
  
## <a name="requirements"></a>Requisitos  
 jsrt.h
