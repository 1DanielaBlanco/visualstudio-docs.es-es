# JsRuntimeVersion (Enumeraci&#243;n)
Versión del runtime.  
  
## Sintaxis  
  
```  
enum JsRuntimeVersion;  
```  
  
## Miembros  
  
### Valores  
  
|Nombre|Descripción|  
|------------|-----------------|  
|`JsRuntimeVersion10`|Crear runtime con la versión IE10.|  
|`JsRuntimeVersion11`|Crear runtime con la versión IE11.|  
|`JsRuntimeVersionEdge`|Crear runtime con la versión más reciente presente en el equipo en tiempo de ejecución.|  
  
## Requisitos  
 **Encabezado:** jsrt.h
