# <a name="jsvalueref-typedef"></a>JsValueRef (Typedef)
Referencia a un valor de JavaScript.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
typedef JsRef JsValueRef;  
```  
  
## <a name="remarks"></a>Comentarios  
 Un valor de JavaScript es uno de los siguientes tipos de valores: sin definir, Null, booleano, cadena, número u objeto.  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** jsrt.h
