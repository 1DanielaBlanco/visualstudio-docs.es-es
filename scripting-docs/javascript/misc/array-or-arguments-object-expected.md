# Se esperaba la matriz o el objeto arguments
No proporcionaste una matriz como argumento.  Este error solo se aplica al método **Function.prototype.apply**.  Si se especifica, el segundo argumento de esta función debe ser un objeto `Array` o **Arguments**.  
  
### Para corregir este error  
  
-   Especifica un objeto `Array` o **Arguments** como segundo argumento.
