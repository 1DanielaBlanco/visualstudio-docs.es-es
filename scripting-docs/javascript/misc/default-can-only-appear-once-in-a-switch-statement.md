# &#39;default&#39; solo puede aparecer una vez en una instrucci&#243;n &#39;switch&#39;.
Has intentado usar la instrucción **default** más de una vez en una instrucción switch.  La instrucción case default es siempre la última instrucción case en una instrucción switch \(de paso explícito\).  
  
### Para corregir este error  
  
-   Quita las instrucciones case **default** adicionales de la instrucción `switch` \(usa como máximo una instrucción case default en la instrucción switch\).
