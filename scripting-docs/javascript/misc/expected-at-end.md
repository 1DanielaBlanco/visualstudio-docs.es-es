# Se esperaba &#39;@end&#39;
Has intentado crear un bloque de código compilado de forma condicional, pero no has incluido la instrucción **@end** al final.  Las instrucciones de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] se pueden compilar de forma condicional si las escribes dentro de un bloque **@if\/@end**.  
  
### Para corregir este error  
  
-   Agrega la instrucción **@end** correspondiente.
