# Se esperaba una constante
Has intentado usar una variable \(compilación no condicional\) en una instrucción de prueba de compilación condicional.  Las instrucciones de prueba de compilación condicional se deben evaluar como una constante.  
  
### Para corregir este error  
  
-   Reemplaza la variable con un literal.  
  
-   Reemplaza la variable con una variable de compilación condicional.
