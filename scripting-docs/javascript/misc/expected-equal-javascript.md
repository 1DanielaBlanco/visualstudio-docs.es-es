# Se esperaba &#39;=&#39; (JavaScript)
Has intentado crear una variable que se usará con instrucciones de compilación condicional, pero no has puesto un signo igual entre la variable y el valor que le deseas asignar.  
  
### Para corregir este error  
  
-   Agrega un signo igual.  Por ejemplo:  
  
    ```javascript  
    @set @myvar1 = 1  
    ```
