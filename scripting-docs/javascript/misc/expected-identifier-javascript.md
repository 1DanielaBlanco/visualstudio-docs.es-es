# Se esperaba un identificador (JavaScript)
Has usado algo distinto de un identificador en un contexto donde se necesitaba uno.  Un identificador puede ser:  
  
-   una variable  
  
-   una propiedad  
  
-   una matriz  
  
-   o un nombre de función.  
  
### Para corregir este error  
  
-   Cambia la expresión para que aparezca un identificador a la izquierda del signo igual.
