# Se esperaba &#39;]&#39;
Has hecho una referencia a un elemento de matriz, pero no incluía el corchete de cierre.  Cualquier expresión que hace referencia a un elemento de matriz debe incluir tanto corchetes de apertura como de cierre.  
  
### Para corregir este error  
  
-   Agrega el corchete de cierre a la expresión que hace referencia al elemento de matriz.
