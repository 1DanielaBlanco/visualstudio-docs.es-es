# Se esperaba &#39;;&#39;
Has intentado colocar más de una instrucción en una línea, pero no las has separado con signos de punto y coma.  Los signos de punto y coma se utilizan para finalizar instrucciones.  Aunque puedes colocar varias instrucciones en una sola línea, debes delimitar cada una de ellas de la siguiente con un punto y coma.  
  
### Para corregir este error  
  
-   Marca el final de cada instrucción con un punto y coma.  
  
-   Asegúrate de usar correctamente los paréntesis en las llamadas de función.  
  
-   Asegúrate de no olvidarte de poner signos de punto y coma dentro del encabezado de un bucle **for**.
