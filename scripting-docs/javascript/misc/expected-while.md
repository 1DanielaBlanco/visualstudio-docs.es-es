# Se esperaba &#39;while &#39;
No has incluido la condición `while` en un bucle `do...while`.  Una instrucción **do** debe tener una prueba `while` correspondiente al final del bloque de código.  
  
### Para corregir este error  
  
-   Incluye la instrucción de prueba `while` después de la llave de cierre.
