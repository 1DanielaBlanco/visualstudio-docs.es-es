# Asignaci&#243;n no v&#225;lida (JavaScript)
Has intentado asignar un valor a un identificador de solo lectura.  No puedes asignar un valor a un identificador de solo lectura.  Por ejemplo, objetos y objetos COM externos definidos por el host.  
  
### Para corregir este error  
  
-   No asignes valores a identificadores de solo lectura.
