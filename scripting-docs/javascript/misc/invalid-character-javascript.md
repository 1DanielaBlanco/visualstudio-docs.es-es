# Car&#225;cter no v&#225;lido (JavaScript)
Ha compuesto un identificador usando un carácter o varios caracteres que el compilador de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] no reconoce como válidos.  Los caracteres válidos siguen estas reglas:  
  
-   El primer carácter debe ser una letra ASCII \(en mayúsculas o minúsculas\) o un carácter de subrayado \(\_\).  
  
-   Los siguientes caracteres pueden ser letras ASCII, números o caracteres de subrayado.  
  
-   El nombre de identificador no puede ser una palabra reservada.  
  
### Para corregir este error  
  
-   Evite usar caracteres que no formen parte de la definición de lenguaje [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
