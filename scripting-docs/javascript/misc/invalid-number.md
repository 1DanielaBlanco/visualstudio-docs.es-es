# N&#250;mero no v&#225;lido
Número no válido \(ErrbadNumber 1013\)  
  
 Has intentado usar un número que no es válido en JavaScript.  
  
 Para obtener información sobre los números válidos, consulta [Number \(Objeto\)](../../javascript/reference/number-object-javascript.md).
