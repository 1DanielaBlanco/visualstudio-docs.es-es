# Se esperaba un objeto de JavaScript
Has intentado pasar un objeto que no es de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] a una función integrada que espera un objeto de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  Las diversas funciones integradas necesitan objetos definidos en [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] \(frente a los objetos definidos por el host, o un componente externo como un control\).  
  
### Para corregir este error  
  
-   Asegúrate de que el objeto que pasas como parámetro es del tipo correcto.
