# No se encuentra la etiqueta
Has creado una referencia a una etiqueta que no existe.  Las etiquetas se pueden usar para marcar bloques de código pero, dentro de un ámbito especificado, deben ser únicas.  
  
### Para corregir este error  
  
-   Asegúrate de que no has escrito incorrectamente el nombre de etiqueta.  
  
-   Asegúrate de que todas las referencias de etiqueta se realizan en etiquetas que se han definido en el ámbito actual \(esto incluye definiciones adelantadas\).
