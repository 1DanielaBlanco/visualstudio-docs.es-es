# Etiqueta redefinida
Has creado una nueva etiqueta, pero le has asignado el nombre de una etiqueta existente.  Las etiquetas se pueden usar para marcar bloques de código pero, dentro de un ámbito especificado, deben ser únicas.  
  
### Para corregir este error  
  
-   Asegúrate de que todas las etiquetas que usas en los programas sean únicas dentro de sus ámbitos respectivos.
