# Se esperaba una cadena.
Has intentado invocar el método **String.prototype.toString** o **String.prototype.valueOf** en un objeto de un tipo distinto de `String`.  El objeto de este tipo de invocación debe ser de tipo `String`.  
  
### Para corregir este error  
  
-   Invoca solo los métodos **String.prototype.toString** o **String.prototype.valueOf** en objetos de tipo `String`.
