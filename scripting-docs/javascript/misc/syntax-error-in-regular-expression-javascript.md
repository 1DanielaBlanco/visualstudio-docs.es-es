# <a name="syntax-error-in-regular-expression-javascript"></a>Error de sintaxis en expresión regular (JavaScript)
La estructura de la cadena de búsqueda infringe una o varias de las reglas gramaticales de una [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] expresión regular.  
  
### <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Garantizar la estructura de la cadena de búsqueda de expresión regular se ajusta a la [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] sintaxis de expresiones regulares.
