# Error de sintaxis en expresi&#243;n regular (JavaScript)
La estructura de la cadena de búsqueda infringe una o más reglas gramaticales de una expresión regular de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
### Para corregir este error  
  
-   Asegúrate de que la estructura de la cadena de búsqueda de expresión regular se adhiere a la sintaxis de expresiones regulares de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
