# Error de sintaxis (JavaScript)
Has creado una instrucción que infringe una o más reglas gramaticales de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].  
  
### Para corregir este error  
  
-   Vuelve a comprobar la sintaxis del programa en el número de línea indicado.  
  
-   Busca si hay paréntesis o llaves colocados erróneamente.
