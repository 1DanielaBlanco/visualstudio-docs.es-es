# La precisi&#243;n est&#225; fuera del intervalo
Has intentado pasar un argumento no válido a la función **Number.prototype.toPrecision**.  El argumento de **toPrecision** debe entre 1 y 21 \(incluidos\).  
  
### Para corregir este error  
  
-   Asegúrate de que el argumento de `toPrecision` no es demasiado grande o demasiado pequeño.
