# Comentario sin terminar
Has iniciado un bloque de comentario de varias líneas, pero no lo has finalizado correctamente.  Los comentarios de varias líneas comienzan con la combinación "\/\*" y finalizan con la combinación inversa "\*\/".  A continuación se muestra un ejemplo:  
  
```javascript  
/* This is a comment  
This is another part of the same comment.*/  
```  
  
### Para corregir este error  
  
-   Asegúrate de terminar los comentarios de varias líneas con "\*\/".
