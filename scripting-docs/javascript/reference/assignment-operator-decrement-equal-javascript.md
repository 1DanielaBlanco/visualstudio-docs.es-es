# <a name="assignment-operator--javascript"></a>Operador de asignación (=) (JavaScript)
Asigna un valor a una variable.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parámetros  
 `result`  
 Cualquier variable.  
  
 `expression`  
 Cualquier expresión numérica.  
  
## <a name="remarks"></a>Comentarios  
 El = operador se comporta igual que otros operadores, por lo que las expresiones que lo contienen tienen un valor. Esto significa que puede encadenar operadores de asignación como sigue: `j = k = l = 0`. En este caso `j`, `k`, y `l` son iguales a cero.
