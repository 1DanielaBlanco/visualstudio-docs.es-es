# Operador de asignaci&#243;n (=) (JavaScript)
Asigna un valor a una variable.  
  
## Sintaxis  
  
```  
  
result = expression  
```  
  
## Parámetros  
 `result`  
 Cualquier variable.  
  
 `expression`  
 Cualquier expresión numérica.  
  
## Comentarios  
 El operador \= se comporta como otros operadores, por lo que las expresiones que lo contienen tienen un valor.  Esto significa que puedes encadenar operadores de asignación del siguiente modo: `j = k = l = 0`.  En este caso `j`, `k` y `l` son igual a cero.
