# <a name="clear-method-set-javascript"></a>clear (Método, Set de JavaScript)
Quita todos los elementos del conjunto.  
  
## <a name="syntax"></a>Sintaxis  
  
```JavaScript  
setObj.clear()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `setObj`  
 Obligatorio. El conjunto para borrar.
