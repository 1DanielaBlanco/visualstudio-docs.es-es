# Debug.debuggerEnabled (Propiedad)
Determina si la depuración está habilitada para el contexto de script.  La depuración se puede habilitar o deshabilitar aunque no haya un depurador adjunto.  
  
## Sintaxis  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
