# Operador de divisi&#243;n (/) (JavaScript)
Divide el valor de dos expresiones.  
  
## Sintaxis  
  
```  
  
result = number1 / number2   
```  
  
## Parámetros  
 `result`  
 Cualquier variable numérica.  
  
 `number1`  
 Cualquier expresión numérica.  
  
 `number2`  
 Cualquier expresión numérica.
