# <a name="getday-method-date-javascript"></a>getDay (Método, Date de JavaScript)
Obtiene el día de la semana, mediante la hora local.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
dateObj. getDay()   
```  
  
#### <a name="parameters"></a>Parámetros  
 La referencia a `dateObj` necesaria es un objeto `Date` .  
  
## <a name="return-value"></a>Valor devuelto  
 Un entero entre 0 y 6 que representa el día de la semana (el domingo es 0, el sábado es 6).  
  
## <a name="remarks"></a>Comentarios  
 Para obtener el día mediante la hora Universal coordinada (UTC), use el `getUTCDay` método.  
  
 En el siguiente ejemplo, se muestra cómo utilizar el método `getDay`.  
  
```JavaScript  
var date = new Date("Saturday, February 9, 2008");  
day = date.getDay();  
document.write(day);  
  
// Output: 6  
  
```
