# <a name="global-object-javascript"></a>Global (Objeto de JavaScript)
Un objeto intrínseco cuyo propósito es recopilar constantes y funciones globales en un objeto.  
  
## <a name="remarks"></a>Comentarios  
 El `Global` objeto nunca se utiliza directamente y no se puede crear mediante el `new` operador. Se crea cuando se inicializa el motor de scripting, lo que hace que sus funciones y constantes disponibles inmediatamente.
