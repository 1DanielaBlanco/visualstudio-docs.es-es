# Infinity (Constante de JavaScript)
Un número que es mayor que el mayor número de punto flotante.  Devuelve el valor inicial de `Number.POSITIVE_INFINITY`.  Infinito negativo \(\-Infinity\) es menor que el menor número de punto flotante.  
  
## Sintaxis  
  
```  
Infinity   
```  
  
## Comentarios  
 La constante `Infinity` es un miembro del objeto `Global` y está disponible cuando se inicializa el motor de scripting.
