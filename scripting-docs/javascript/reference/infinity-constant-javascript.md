# <a name="infinity-constant-javascript"></a>Infinity (Constante de JavaScript)
Un número que es mayor que el mayor número de punto flotante. Devuelve un valor inicial de `Number.POSITIVE_INFINITY`. Infinito negativo (-infinito) es menor que el flotante más pequeño número de punto.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
Infinity   
```  
  
## <a name="remarks"></a>Comentarios  
 El `Infinity` constante es un miembro de la `Global` del objeto y está disponible cuando se inicializa el motor de scripting.
