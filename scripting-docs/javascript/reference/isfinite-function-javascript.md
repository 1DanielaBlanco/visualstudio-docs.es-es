# <a name="isfinite-function-javascript"></a>isFinite (Función de JavaScript)
Determina si un número proporcionado es finito.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
isFinite(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `number` argumento es cualquier valor numérico.  
  
 El `isFinite` función devuelve `true` si `number` es cualquier valor distinto de `NaN`, infinito negativo o infinito positivo. En los tres casos, devuelve **false**.
