# <a name="javascript-future-reserved-words"></a>Futuras palabras reservadas de JavaScript
En JavaScript, futuras palabras clave reservadas, no debe usarse como identificadores, aunque no tienen ningún significado especial en la versión actual.  
  
 Para obtener una lista de palabras reservadas de JavaScript, consulte [palabras reservadas de JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## <a name="future-reserved-words"></a>Futuras palabras reservadas  
 Éstas son palabras reservadas para el futuro:  
  
1.  clase  
  
2.  enum  
  
3.  exportar  
  
4.  extiende  
  
5.  importación  
  
6.  Super  
  
## <a name="future-reserved-words-in-strict-mode"></a>Futuras palabras reservadas en modo strict  
 Éstos son futuras palabras reservadas solo en modo strict. Para obtener más información sobre el modo strict, consulte [directiva use strict](../../javascript/reference/use-strict-directive.md).  
  
1.  implementa  
  
2.  interfaz  
  
3.  paquete  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
