# Futuras palabras reservadas de JavaScript
En JavaScript, las palabras clave reservadas para uso futuro no se deben usar como identificadores, aunque no tienen ningún significado especial en la versión actual.  
  
 Para obtener una lista de palabras reservadas en JavaScript, vea [Palabras reservadas de JavaScript](../../javascript/reference/javascript-reserved-words.md).  
  
## Futuras palabras reservadas  
 Las siguientes son palabras reservadas para uso futuro:  
  
1.  clase  
  
2.  enum  
  
3.  export  
  
4.  extends  
  
5.  import  
  
6.  super  
  
## Palabras reservadas para uso futuro en modo strict  
 Las siguientes son palabras reservadas para uso futuro solo en modo strict.  Para obtener más información sobre el modo strict, vea [Directiva use strict](../../javascript/reference/use-strict-directive.md).  
  
1.  implements  
  
2.  interfaz  
  
3.  package  
  
4.  private  
  
5.  protected  
  
6.  public  
  
7.  static  
  
8.  yield
