# Palabras reservadas de JavaScript
[!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] tiene varias palabras reservadas que no se pueden usar como identificadores.  Las palabras reservadas tienen un significado específico para el lenguaje [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)], ya que forman parte de la sintaxis del lenguaje.  El uso de una palabra reservada produce un error de compilación al cargar el script.  
  
 [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] también cuenta con una lista de palabras reservadas para uso futuro.  Estas palabras no forman parte actualmente del lenguaje [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)], aunque están reservadas para uso futuro.  Para obtener más información sobre las palabras clave reservadas para uso futuro en JavaScript, vea [Futuras palabras reservadas de JavaScript](../../javascript/reference/javascript-future-reserved-words.md).  
  
 A la hora de elegir identificadores también deberá evitarse cualquier palabra que ya sea nombres de objetos o funciones intrínsecas de [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)], como `String` o `parseInt`.
