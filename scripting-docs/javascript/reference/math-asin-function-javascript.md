# Math.asin (Funci&#243;n de JavaScript)
Devuelve el arcoseno de un número.  
  
## Sintaxis  
  
```  
  
Math.asin(number)   
```  
  
## Comentarios  
 El argumento obligatorio `number` es una expresión numérica cuyo arcoseno deseas obtener.  
  
 El valor devuelto es el arcoseno del argumento `number`, en radianes.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
