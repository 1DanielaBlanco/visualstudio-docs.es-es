# <a name="mathasin-function-javascript"></a>Math.asin (Función de JavaScript)
Devuelve el arco seno de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.asin(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `number` argumento es una expresión numérica para la que se necesita el arcoseno.  
  
 El valor devuelto es el arcoseno de los `number` argumento, en radianes.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
