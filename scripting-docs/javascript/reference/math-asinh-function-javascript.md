# <a name="mathasinh-function-javascript"></a>Math.asinh (función de JavaScript)
Devuelve el seno hiperbólico inverso de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>Comentarios  
 Si `x` es `NaN`, el resultado es `NaN`.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
