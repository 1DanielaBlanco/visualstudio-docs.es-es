# <a name="mathatan-function-javascript"></a>Math.atan (Función de JavaScript)
Devuelve el arco tangente de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.atan(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `number` argumento es una expresión numérica para la que se necesita el arco tangente.  
  
 El valor devuelto es el arco tangente de la `number` argumento, en radianes.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
