# Math.atan (Funci&#243;n de JavaScript)
Devuelve el arco tangente de un número.  
  
## Sintaxis  
  
```  
  
Math.atan(number)   
```  
  
## Comentarios  
 El argumento `number` necesario es una expresión numérica cuyo arco tangente se desea obtener.  
  
 El valor devuelto es el arco tangente del argumento `number`, en radianes.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
