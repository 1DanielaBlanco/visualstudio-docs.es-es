# <a name="mathatan2-function-javascript"></a>Math.atan2 (Función de JavaScript)
Devuelve el ángulo (en radianes) desde el eje X hasta un punto (*y*,*x*).  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.atan2(y, x)  
```  
  
## <a name="parameters"></a>Parámetros  
 *x*  
 Obligatorio. Una expresión numérica que representa la coordenada x cartesiana.  
  
 *y*  
 Obligatorio. Una expresión numérica que representa la coordenada y cartesiana.  
  
## <a name="remarks"></a>Comentarios  
 El valor devuelto está comprendido entre *-pi* y *pi*. Representa el ángulo de proporcionado (*y*,*x*) seleccione, en radianes.
