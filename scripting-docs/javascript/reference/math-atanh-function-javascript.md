# <a name="mathatanh-function-javascript"></a>Math.atanh (función de JavaScript)
Devuelve la tangente hiperbólica inversa de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.atanh(x)   
```  
  
## <a name="remarks"></a>Comentarios  
 Si `x` es `NaN`, el resultado es `NaN`.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
