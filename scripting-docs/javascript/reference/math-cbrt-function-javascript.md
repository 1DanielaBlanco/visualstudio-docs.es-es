# <a name="mathcbrt-function-javascript"></a>Math.cbrt (función de JavaScript)
Devuelve la raíz cúbica de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.cbrt(  
number  
)   
```  
  
## <a name="remarks"></a>Comentarios  
 Si `number` es `NaN`, el resultado será `NaN`.
