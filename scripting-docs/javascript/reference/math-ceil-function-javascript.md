# <a name="mathceil-function-javascript"></a>Math.ceil (Función de JavaScript)
Devuelve el entero más pequeño mayor o igual que su argumento numérico.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.ceil(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 El argumento `number` necesario es una expresión numérica.  
  
 El valor devuelto es un valor entero igual al entero más pequeño mayor o igual que su argumento numérico.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
