# <a name="mathcos-function-javascript"></a>Math.cos (Función de JavaScript)
Devuelve el coseno de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.cos(radians)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `radians` argumento es una expresión numérica que contiene un ángulo, medido en radianes.  
  
 El valor devuelto es el coseno del argumento numérico del `radians`.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
