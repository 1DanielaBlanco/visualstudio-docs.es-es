# Math.cos (Funci&#243;n de JavaScript)
Devuelve el coseno de un número.  
  
## Sintaxis  
  
```  
  
Math.cos(radians)   
```  
  
## Comentarios  
 El argumento obligatorio `radians` es una expresión numérica que contiene un ángulo medido en radianes.  
  
 El valor devuelto es el coseno del argumento numérico de `radians`.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
