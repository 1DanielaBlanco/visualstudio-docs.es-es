# Math.cosh (funci&#243;n de JavaScript)
Devuelve el coseno hiperbólico de un número.  
  
## Sintaxis  
  
```  
  
Math.cosh(x)   
```  
  
## Comentarios  
 El valor devuelto es el mismo que `(exp(x) + exp(-x))/2`.  Si `x` es `NaN`, el resultado es `NaN`.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
