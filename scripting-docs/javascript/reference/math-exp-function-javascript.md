# <a name="mathexp-function-javascript"></a>Math.exp (Función de JavaScript)
Devuelve e (la base de los logaritmos naturales) elevado a una potencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `number` argumento es una expresión numérica que representa la potencia de *e*.  
  
 El valor devuelto es un número. La constante *e* es de Euler, aproximadamente igual a 2,71828, número y `number` es el argumento proporcionado.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
