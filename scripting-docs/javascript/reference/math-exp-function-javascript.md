# Math.exp (Funci&#243;n de JavaScript)
Devuelve e \(la base de los logaritmos naturales\) elevado a una potencia.  
  
## Sintaxis  
  
```  
  
Math.exp(number)   
```  
  
## Comentarios  
 El argumento obligatorio`number` es una expresión numérica que representa la potencia de *e*.  
  
 El valor devuelto es un número.  La constante *e* es el número de Euler, que es aproximadamente igual a 2,71828, y `number` es el argumento proporcionado.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
