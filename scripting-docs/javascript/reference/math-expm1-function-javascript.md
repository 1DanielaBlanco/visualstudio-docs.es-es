# <a name="mathexpm1-function-javascript"></a>Math.expm1 (función de JavaScript)
Devuelve el resultado de restar 1 de e (la base de los logaritmos naturales) elevado a una potencia.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.expm1(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 El argumento obligatorio `number` es una expresión numérica que representa la potencia de e.  
  
 El valor devuelto es un número. La constante e es el número de Euler, aproximadamente igual a 2,71828, y `number` es el argumento proporcionado.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
