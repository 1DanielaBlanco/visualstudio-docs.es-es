# <a name="mathfloor-function-javascript"></a>Math.floor (Función de JavaScript)
Devuelve al entero más grande menor o igual que su argumento numérico.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.floor(number)   
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido *número* argumento es una expresión numérica.  
  
 El valor devuelto es un valor entero igual al entero más grande menor o igual que su argumento numérico.  
  
 **Se aplica a**: [Math (objeto)](../../javascript/reference/math-object-javascript.md)
