# Math.floor (Funci&#243;n de JavaScript)
Devuelve el mayor entero que sea menor o igual que su argumento numérico.  
  
## Sintaxis  
  
```  
  
Math.floor(number)   
```  
  
## Comentarios  
 El argumento *number* necesario es una expresión numérica.  
  
 El valor devuelto es un valor entero igual al mayor entero que sea menor o igual que su argumento numérico.  
  
 **Se aplica a**: [Math \(Objeto\)](../../javascript/reference/math-object-javascript.md)
