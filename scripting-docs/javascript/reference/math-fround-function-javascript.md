# Math.fround (funci&#243;n de JavaScript)
Devuelve el formato de punto flotante de precisión simple más cercano de un número.  
  
## Sintaxis  
  
```  
  
Math.fround(  
number  
)   
```  
  
## Comentarios  
 Si `number` es `NaN`, el resultado es `NaN`.
