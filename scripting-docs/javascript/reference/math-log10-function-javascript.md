# <a name="mathlog10-function-javascript"></a>Math.log10 (función de JavaScript)
Devuelve el logaritmo en base 10 de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
Math.log10(x)   
```  
  
## <a name="return-value"></a>Valor devuelto  
 Si `x` es `NaN`, esta función devuelve `NaN`. Si `x` es menor que cero (0), esta función devuelve `NaN`.
