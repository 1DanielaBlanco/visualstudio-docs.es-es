# Math.log2 (funci&#243;n de JavaScript)
Devuelve el logaritmo en base 2 de un número.  
  
## Sintaxis  
  
```  
Math.log2(x)   
```  
  
## Valor devuelto  
 Si `x` es `NaN`, esta función devuelve `NaN`.  Si `x` es menor que cero \(0\), esta función devuelve `NaN`.
