# <a name="mathpow-function-javascript"></a>Math.pow (Función de JavaScript)
Devuelve el valor de una expresión base realizada a una potencia especificada.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>Parámetros  
 *base*  
 Obligatorio. El valor base de la expresión.  
  
 *exponente*  
 Obligatorio. El valor de exponente de la expresión.  
  
## <a name="example"></a>Ejemplo  
 En el ejemplo siguiente, una expresión numérica igual a baseexponent devuelve 1000.  
  
```JavaScript  
Math.pow(10,3);  
```
