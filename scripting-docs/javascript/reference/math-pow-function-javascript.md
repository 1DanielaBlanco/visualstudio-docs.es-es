# Math.pow (Funci&#243;n de JavaScript)
Devuelve el valor de una expresión formada por una base elevada a una potencia especificada.  
  
## Sintaxis  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## Parámetros  
 *base*  
 Obligatorio.  Valor de la base de la expresión.  
  
 *exponent*  
 Obligatorio.  Valor del exponente de la expresión.  
  
## Ejemplo  
 En el ejemplo siguiente, una expresión numérica igual a baseexponent devuelve 1000.  
  
```javascript  
Math.pow(10,3);  
```
