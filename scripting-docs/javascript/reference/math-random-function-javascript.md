# <a name="mathrandom-function-javascript"></a>Math.random (Función de JavaScript)
Devuelve un número pseudoaleatorio entre 0 y 1.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.random( )  
  
```  
  
## <a name="remarks"></a>Comentarios  
 El número pseudoaleatorio generado está comprendido entre 0 (inclusive) y 1 (excluido), es decir, el número devuelto puede ser cero, pero siempre será menor que uno. El generador de números aleatorios se inicializa automáticamente cuando [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] se carga por primera vez.
