# Math.random (Funci&#243;n de JavaScript)
Devuelve un número seudoaleatorio entre 0 y 1.  
  
## Sintaxis  
  
```  
  
Math.random( )  
  
```  
  
## Comentarios  
 El número seudoaleatorio generado está comprendido entre 0 \(incluido\) y 1 \(excluido\); es decir, el número devuelto puede ser cero, pero siempre será menor que uno.  El generador de números aleatorios se inicializa automáticamente la primera vez que se carga [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)].
