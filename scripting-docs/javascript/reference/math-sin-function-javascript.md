# <a name="mathsin-function-javascript"></a>Math.sin (Función de JavaScript)
Devuelve el seno de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.sin(radians)   
```  
  
## <a name="remarks"></a>Comentarios  
 El `radians` argumento es una expresión numérica que contiene un ángulo, medido en radianes.  
  
 El valor devuelto es el seno del argumento numérico.
