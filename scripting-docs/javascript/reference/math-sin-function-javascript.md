# Math.sin (Funci&#243;n de JavaScript)
Devuelve el seno de un número.  
  
## Sintaxis  
  
```  
  
Math.sin(radians)   
```  
  
## Comentarios  
 El argumento `radians` es una expresión numérica que contiene un ángulo medido en radianes.  
  
 El valor devuelto es el seno del argumento numérico.
