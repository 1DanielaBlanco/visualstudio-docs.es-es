# <a name="mathsinh-function-javascript"></a>Math.sinh (función de JavaScript)
Devuelve el seno hiperbólico inverso de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.sinh(x)   
```  
  
## <a name="remarks"></a>Comentarios  
 El valor devuelto es el mismo que `(exp(x) - exp(-x))/2`. Si x es `NaN`, el resultado es `NaN`.
