# Math.sinh (funci&#243;n de JavaScript)
Devuelve el seno hiperbólico inverso de un número.  
  
## Sintaxis  
  
```  
  
Math.sinh(x)   
```  
  
## Comentarios  
 El valor devuelto es el mismo que `(exp(x) - exp(-x))/2`.  Si x es `NaN`, el resultado es `NaN`.
