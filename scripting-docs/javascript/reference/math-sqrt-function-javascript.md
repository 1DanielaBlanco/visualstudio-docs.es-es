# Math.sqrt (Funci&#243;n de JavaScript)
Devuelve la raíz cuadrada de un número.  
  
## Sintaxis  
  
```  
  
Math.sqrt(  
number  
)   
```  
  
## Comentarios  
 El argumento `number` necesario es una expresión numérica.  
  
 Si `number` es negativo, el valor devuelto es `NaN`.
