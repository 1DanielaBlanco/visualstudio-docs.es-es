# <a name="mathtan-function-javascript"></a>Math.tan (Función de JavaScript)
Devuelve la tangente de un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido `radians` argumento es una expresión numérica que contiene un ángulo, medido en radianes.  
  
 El valor devuelto es la tangente del argumento numérico del `radians`.
