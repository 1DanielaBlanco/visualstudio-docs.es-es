# Math.tan (Funci&#243;n de JavaScript)
Devuelve la tangente de un número.  
  
## Sintaxis  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## Comentarios  
 El argumento obligatorio `radians` es una expresión numérica que contiene un ángulo medido en radianes.  
  
 El valor devuelto es la tangente del argumento numérico de `radians`.
