# Math.tanh (funci&#243;n de JavaScript)
Devuelve la tangente hiperbólica de un número.  
  
## Sintaxis  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## Comentarios  
 El valor devuelto es el mismo que `(exp(x) - exp(-x))/(exp(x) + exp(-x))`.  Si `x` es `NaN`, el resultado es `NaN`.
