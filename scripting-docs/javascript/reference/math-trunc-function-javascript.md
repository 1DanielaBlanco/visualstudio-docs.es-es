# <a name="mathtrunc-function-javascript"></a>Función Math.trunc (JavaScript)
Devuelve la parte entera de un número, quitando los dígitos fraccionarios.  
  
## <a name="syntax"></a>Sintaxis  
  
```vb  
Math.trunc(number)  
```  
  
## <a name="remarks"></a>Comentarios  
 El argumento obligatorio `number` es una expresión numérica para la que se necesita el entero truncado.  
  
 Si `number` es `NaN`, devuelve `NaN`.
