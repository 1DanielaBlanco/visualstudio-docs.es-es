# Operador de multiplicaci&#243;n (*) (JavaScript)
Multiplica el valor de dos expresiones.  
  
## Sintaxis  
  
```  
  
result = number1*number2  
```  
  
## Parámetros  
 *result*  
 Cualquier variable.  
  
 `number1`  
 Cualquier expresión.  
  
 `number2`  
 Cualquier expresión.
