# <a name="nan-constant-javascript"></a>NaN (Constante de JavaScript)
Una constante especial que especifica thatNot una expresión no es un número.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
NaN   
```  
  
## <a name="remarks"></a>Comentarios  
 El `NaN` constante (no un número) es un miembro de la `Global` del objeto y está disponible cuando se inicializa el motor de scripting.
