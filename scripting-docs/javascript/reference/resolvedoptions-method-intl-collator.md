# <a name="resolvedoptions-method-intlcollator"></a>resolvedOptions (Método, Intl.Collator)
Devuelve un objeto que contiene las propiedades y los valores del objeto clasificador.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `collatorObj`  
 La `Collator` objeto que se va a recuperar la información de.  
  
## <a name="remarks"></a>Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas de la `Collator` objeto.
