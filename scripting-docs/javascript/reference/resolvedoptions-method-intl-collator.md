# resolvedOptions (M&#233;todo, Intl.Collator)
Devuelve un objeto que contiene las propiedades y los valores del objeto de intercalador.  
  
## Sintaxis  
  
```  
collatorObj.resolvedOptions()  
```  
  
#### Parámetros  
 `collatorObj`  
 Objeto `Collator`del que se recuperará la información.  
  
## Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas del objeto `Collator`.
