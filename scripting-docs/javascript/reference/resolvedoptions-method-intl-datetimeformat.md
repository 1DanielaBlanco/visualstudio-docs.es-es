# resolvedOptions (M&#233;todo, Intl.DateTimeFormat)
Devuelve un objeto que contiene las propiedades y los valores del objeto del formateador de fecha y hora.  
  
## Sintaxis  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### Parámetros  
 `dateTimeFormatObj`  
 Objeto `DateTimeFormat` del que se recuperará la información.  
  
## Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas del objeto `DateTimeFormat`.
