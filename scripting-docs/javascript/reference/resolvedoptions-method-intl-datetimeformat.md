# <a name="resolvedoptions-method-intldatetimeformat"></a>resolvedOptions (Método, Intl.DateTimeFormat)
Devuelve un objeto que contiene las propiedades y los valores del objeto del formateador de fecha y hora.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
dateTimeFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `dateTimeFormatObj`  
 La `DateTimeFormat` objeto que se va a recuperar la información de.  
  
## <a name="remarks"></a>Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas de la `DateTimeFormat` objeto.
