# <a name="resolvedoptions-method-intlnumberformat"></a>resolvedOptions (Método, Intl.NumberFormat)
Devuelve un objeto que contiene las propiedades y los valores de la `NumberFormat` objeto de formateador.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `numberFormatObj`  
 La `NumberFormat` objeto que se va a recuperar la información de.  
  
## <a name="remarks"></a>Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas de la `NumberFormat` objeto.
