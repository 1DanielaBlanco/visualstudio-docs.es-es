# resolvedOptions (M&#233;todo, Intl.NumberFormat)
Devuelve un objeto que contiene las propiedades y los valores del objeto `NumberFormat` del formateador.  
  
## Sintaxis  
  
```  
numberFormatObj.resolvedOptions()  
```  
  
#### Parámetros  
 `numberFormatObj`  
 Objeto `NumberFormat` del que se recuperará la información.  
  
## Comentarios  
 Las propiedades del objeto devuelto corresponden a las propiedades calculadas del objeto `NumberFormat`.
