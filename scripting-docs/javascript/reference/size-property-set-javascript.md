# size (Propiedad, Set de JavaScript)
Devuelve el número de elementos de un conjunto.  
  
## Sintaxis  
  
```javascript  
sizeVar = setObj.size  
```  
  
#### Parámetros  
 `sizeVar`  
 Requerido.  Cualquier número.  
  
 `setObj`  
 Requerido.  Cualquier objeto `Set`.
