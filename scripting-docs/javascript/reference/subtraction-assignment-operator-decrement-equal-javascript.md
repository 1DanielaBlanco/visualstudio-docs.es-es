# Operador de resta y asignaci&#243;n (-=) (JavaScript)
Resta el valor de una expresión del valor de una variable y asigna el resultado a la variable.  
  
## Sintaxis  
  
```  
  
result -= expression  
```  
  
## Parámetros  
 *result*  
 Cualquier variable numérica.  
  
 *expression*  
 Cualquier expresión numérica.  
  
## Comentarios  
 El uso del operador `-=` es exactamente igual que hacer lo siguiente:  
  
```javascript  
result = result – expression  
```
