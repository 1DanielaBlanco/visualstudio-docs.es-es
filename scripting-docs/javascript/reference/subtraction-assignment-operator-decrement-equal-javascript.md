# <a name="subtraction-assignment-operator---javascript"></a>Operador de resta y asignación (-=) (JavaScript)
Resta el valor de una expresión del valor de una variable y asigna el resultado a la variable.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
result -= expression  
```  
  
## <a name="parameters"></a>Parámetros  
 *resultado*  
 Cualquier variable numérica.  
  
 *expresión*  
 Cualquier expresión numérica.  
  
## <a name="remarks"></a>Comentarios  
 Mediante el `-=` operador es exactamente igual que las acciones siguientes:  
  
```JavaScript  
result = result - expression  
```
