# <a name="todatestring-method-date-javascript"></a>toDateString (Método, Date de JavaScript)
Devuelve una fecha como un valor de cadena.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
objDate.toDateString( )  
```  
  
## <a name="remarks"></a>Comentarios  
 Requerido *objDate* referencia es un `Date` objeto.  
  
 El `toDateString` método devuelve un valor de cadena que contiene la fecha, en la zona horaria actual, en un formato de fácil lectura.
