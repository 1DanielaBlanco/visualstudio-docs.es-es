# toDateString (M&#233;todo, Date de JavaScript)
Devuelve una fecha como un valor alfanumérico.  
  
## Sintaxis  
  
```  
  
objDate.toDateString( )  
```  
  
## Comentarios  
 La referencia *objDate* obligatoria es un objeto `Date`.  
  
 El método `toDateString` devuelve un valor alfanumérico que contiene la fecha, en la zona horaria actual, en un formato cómodo de fácil lectura.
