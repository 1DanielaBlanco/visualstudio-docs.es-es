# <a name="tostring-method-map-javascript"></a>toString (Método, Map de JavaScript)
Devuelve una representación de cadena de un mapa.  
  
## <a name="syntax"></a>Sintaxis  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `mapObj`  
 Obligatorio. Objeto `Map`.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 La representación de cadena del mapa.
