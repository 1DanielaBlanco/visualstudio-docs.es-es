# toString (M&#233;todo, Set de JavaScript)
Devuelve una representación de cadena de un conjunto.  
  
## Sintaxis  
  
```javascript  
setObj.toString()  
```  
  
#### Parámetros  
 `setObj`  
 Requerido.  Un objeto `Set`.  
  
## Valor de propiedad y valor devuelto  
 Representación de cadena del conjunto.
