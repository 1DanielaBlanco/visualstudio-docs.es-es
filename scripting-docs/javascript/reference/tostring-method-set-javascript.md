# <a name="tostring-method-set-javascript"></a>toString (Método, Set de JavaScript)
Devuelve una representación de cadena de un conjunto.  
  
## <a name="syntax"></a>Sintaxis  
  
```JavaScript  
setObj.toString()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `setObj`  
 Obligatorio. Objeto `Set`.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 La representación de cadena del conjunto.
