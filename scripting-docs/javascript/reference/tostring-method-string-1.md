# <a name="tostring-method-string-1"></a>toString (método, String) 1
Devuelve una representación de cadena de una cadena.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>Parámetros  
 `string`  
 Obligatorio. La matriz para representar como una cadena.  
  
## <a name="return-value"></a>Valor devuelto  
 La representación de cadena de la cadena.  
  
## <a name="example"></a>Ejemplo  
 En el ejemplo siguiente se muestra el uso de la **toString** método con una cadena.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
