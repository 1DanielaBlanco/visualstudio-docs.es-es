# toString (M&#233;todo, WeakMap de JavaScript)
Devuelve una representación de cadena de un objeto `WeakMap`.  
  
## Sintaxis  
  
```javascript  
weakmapObj.toString()  
```  
  
#### Parámetros  
 `weakmapObj`  
 Requerido.  Un objeto `WeakMap`.  
  
## Valor de propiedad y valor devuelto  
 Representación de cadena de `WeakMap`.
