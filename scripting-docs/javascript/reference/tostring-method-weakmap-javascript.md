# <a name="tostring-method-weakmap-javascript"></a>toString (Método, WeakMap de JavaScript)
Devuelve una representación de cadena de un `WeakMap` objeto.  
  
## <a name="syntax"></a>Sintaxis  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>Parámetros  
 `weakmapObj`  
 Obligatorio. Objeto `WeakMap`.  
  
## <a name="property-valuereturn-value"></a>Valor de propiedad y valor devuelto  
 Representación de cadena de `WeakMap`.
