# <a name="void-operator-javascript"></a>void (Operador de JavaScript)
Evita que una expresión devuelva un valor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
void expression   
```  
  
## <a name="remarks"></a>Comentarios  
 El `expression` argumento se trata de cualquier [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)] expresión.  
  
 El `void` operador evalúa la expresión y devuelve `undefined`. Es útil en situaciones donde debe evaluar una expresión, pero no desea que los resultados visibles para el resto de la secuencia de comandos.
