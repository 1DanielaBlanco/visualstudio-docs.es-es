# IActiveScriptAuthorProcedure (Interfaz)
Además de los métodos heredados de `IUnknown`, la interfaz `IScriptAuthorProcedure` expone el método siguiente.  
  
## métodos en el orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IActiveScriptAuthorProcedure::ParseProcedureText](../../winscript/reference/iactivescriptauthorprocedure-parseproceduretext.md)|Analiza un procedimiento de código, agregue el texto del procedimiento de código al motor de creación de script, y crea un objeto de `IScriptEntry` correspondiente al procedimiento de código.|
