# IActiveScriptGarbageCollector (Interfaz)
Proporciona un método a la recolección de elementos no utilizados de inicio.  Esta interfaz se debe implementar por los motores de scripts activos que desean limpiar sus recursos.  
  
## Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Inicia la recolección de elementos no utilizados.|
