# IActiveScriptSiteInterruptPoll (Interfaz)
La interfaz de `IActiveScriptSiteInterruptPoll` permite que un host especificar un script que termine.  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `IActiveScriptSiteInterruptPoll` expone el método siguiente.  
  
|Método|Descripción|  
|------------|-----------------|  
|[IActiveScriptSiteInterruptPoll::QueryContinue](../../winscript/reference/iactivescriptsiteinterruptpoll-querycontinue.md)|Permite que un host especificar un script que termine.|
