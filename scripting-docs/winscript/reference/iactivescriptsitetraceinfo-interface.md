# IActiveScriptSiteTraceInfo (Interfaz)
Proporciona un método para enviar información de traza del script.  
  
## Métodos  
  
|||  
|-|-|  
|Método|Descripción|  
|[IActiveScriptSiteTraceInfo::SendScriptTraceInfo \(Método\)](../../winscript/reference/iactivescriptsitetraceinfo-sendscripttraceinfo-method.md)|Envía la información de traza que incluye el tipo de evento, contexto y, en el fragmento de script.|
