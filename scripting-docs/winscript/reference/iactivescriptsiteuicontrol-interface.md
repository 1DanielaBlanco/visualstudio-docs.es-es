# IActiveScriptSiteUIControl (Interfaz)
Proporciona un método para obtener el comportamiento de un control de IU.  
  
## Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[IActiveScriptSiteUIControl::GetUIBehavior \(Método\)](../../winscript/reference/iactivescriptsiteuicontrol-getuibehavior-method.md)|Obtiene [SCRIPTUICHANDLING \(Enumeración\)](../../winscript/reference/scriptuichandling-enumeration.md) que representa la manera en que un control de IU debe controlar.|
