# IActiveScriptTraceInfo::StopScriptTracing (M&#233;todo)
Detiene el seguimiento de script.  
  
## Sintaxis  
  
```  
HRESULT StopScriptTracing(void);   
}  
  
```  
  
#### Parámetros  
 Este método no tiene ningún parámetro.  
  
## Valor devuelto  
 Los valores devueltos posibles de este método son los siguientes:  
  
1.  S\_OK: Correctamente.  
  
2.  E\_NOTIMPL: no implementado.
