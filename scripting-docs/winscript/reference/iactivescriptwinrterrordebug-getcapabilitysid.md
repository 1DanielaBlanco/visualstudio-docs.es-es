# IActiveScriptWinRTErrorDebug::GetCapabilitySid
Devuelve la capacidad SID del error de Windows en tiempo de ejecución, si está disponible.  
  
> [!IMPORTANT]
>  [IActiveScriptWinRTErrorDebug \(Interfaz\)](../../winscript/reference/iactivescriptwinrterrordebug-interface.md) es implementado por PDM v11.0 y mayor.  Encontrado en activdbg100.h.  
  
## Sintaxis  
  
```cpp  
HRESULT GetCapabilitySid([out] BSTR * capabilitySid);  
```  
  
#### Parámetros  
 `capabilitySid`  
 La capacidad SID de error.
