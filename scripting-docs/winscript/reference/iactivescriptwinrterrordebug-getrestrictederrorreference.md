# IActiveScriptWinRTErrorDebug::GetRestrictedErrorReference
Devuelve el error limitado Windows en tiempo de ejecución de referencia, si está disponible.  
  
> [!IMPORTANT]
>  [IActiveScriptWinRTErrorDebug \(Interfaz\)](../../winscript/reference/iactivescriptwinrterrordebug-interface.md) es implementado por PDM v11.0 y mayor.  Encontrado en activdbg100.h.  
  
## Sintaxis  
  
```cpp  
HRESULT GetRestrictedErrorReference([out] BSTR * referenceString);  
```  
  
#### Parámetros  
 `referenceString`  
 \[out\] cadena de error de la referencia de El.
