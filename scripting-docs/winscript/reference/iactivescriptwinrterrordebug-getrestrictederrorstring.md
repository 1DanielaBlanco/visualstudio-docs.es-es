# IActiveScriptWinRTErrorDebug::GetRestrictedErrorString
Devuelve la cadena limitada Windows en tiempo de ejecución del error, si está disponible.  
  
> [!IMPORTANT]
>  [IActiveScriptWinRTErrorDebug \(Interfaz\)](../../winscript/reference/iactivescriptwinrterrordebug-interface.md) es implementado por PDM v11.0 y mayor.  Encontrado en activdbg100.h.  
  
## Sintaxis  
  
```cpp  
HRESULT GetRestrictedErrorString([out] BSTR * errorString);  
  
```  
  
#### Parámetros  
 `errorString`  
 \[out\] cadena limitada el error.
