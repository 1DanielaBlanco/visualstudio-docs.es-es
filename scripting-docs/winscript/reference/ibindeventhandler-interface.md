# IBindEventHandler (Interfaz)
Permite al llamador para proporcionar un objeto que administra un controlador de eventos especificado.  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `IBindEventHandler` expone los métodos siguientes.  
  
|Método|Descripción|  
|------------|-----------------|  
|[IBindEventHandler::BindHandler](../../winscript/reference/ibindeventhandler-bindhandler.md)|Enlaza un evento a un objeto.|
