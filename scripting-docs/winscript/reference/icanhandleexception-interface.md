# ICanHandleException (Interfaz)
Permite al llamador de un motor de script especifica que las excepciones el llamador administran.  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `ICanHandleException` expone los métodos siguientes.  
  
|Método|Descripción|  
|------------|-----------------|  
|[ICanHandleException::CanHandleException](../../winscript/reference/icanhandleexception-canhandleexception.md)|Determina si el llamador del motor de script puede controlar una excepción especificada.|
