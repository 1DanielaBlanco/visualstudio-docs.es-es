# IDebugAsyncOperationCallBack (Interfaz)
Proporciona eventos de estado relacionados con el progreso de una evaluación de la interfaz de `IDebugAsyncOperation` .  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `IDebugAsyncOperationCallBack` expone los métodos siguientes.  
  
|Método|Descripción|  
|------------|-----------------|  
|[IDebugAsyncOperationCallBack::onComplete](../../winscript/reference/idebugasyncoperationcallback-oncomplete.md)|Indica que un resultado está disponible de una operación asincrónica de depuración.|
