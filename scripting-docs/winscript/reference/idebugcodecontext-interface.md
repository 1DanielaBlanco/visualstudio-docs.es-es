# IDebugCodeContext (Interfaz)
Una abstracción que representa una posición en código de la aplicación ejecutable.  
  
 Además de los métodos heredados de `IUnknown`, la interfaz `IDebugCodeContext` expone los métodos siguientes.  
  
## métodos en el orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IDebugCodeContext::GetDocumentContext](../../winscript/reference/idebugcodecontext-getdocumentcontext.md)|Devuelve el contexto del documento asociado a este contexto de código.|  
|[IDebugCodeContext::SetBreakPoint](../../winscript/reference/idebugcodecontext-setbreakpoint.md)|Obtiene o borrar un punto de interrupción en este contexto de código.|
