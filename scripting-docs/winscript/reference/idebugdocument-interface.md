# IDebugDocument (Interfaz)
La interfaz base para todos los documentos de depuración.  
  
## Comentarios  
 La interfaz de `IDebugDocument` expone solo los métodos heredados de [IDebugDocumentInfo \(Interfaz\)](../../winscript/reference/idebugdocumentinfo-interface.md).  No expone ningún método adicional.
