# IDebugExpressionCallBack (Interfaz)
Proporciona eventos de estado relacionados con el progreso de una evaluación de `IDebugExpression` .  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `IDebugExpressionCallBack` expone los métodos siguientes.  
  
|Método|Descripción|  
|------------|-----------------|  
|[IDebugExpressionCallBack::onComplete](../../winscript/reference/idebugexpressioncallback-oncomplete.md)|Indica que la evaluación de la expresión está completa.|
