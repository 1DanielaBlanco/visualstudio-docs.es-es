# IDebugPropertyEnumType_All::GetName
Devuelve un valor BSTR que contiene el nombre de `EnumType`.  
  
## Sintaxis  
  
```  
HRESULT GetName(  
   BSTR*  pname  
);  
```  
  
#### Parámetros  
 `pname`  
 \[out\] En BSTR que contiene el nombre de `EnumType`.  
  
## Valor devuelto  
 Devuelve `HRESULT`válido, normalmente `S_OK`.
