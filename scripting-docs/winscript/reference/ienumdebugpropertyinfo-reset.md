# IEnumDebugPropertyInfo::Reset
Restablece una secuencia de enumeración al principio.  
  
## Sintaxis  
  
```  
HRESULT Reset (  
void  
);  
```  
  
## Valor devuelto  
 Devuelve `HRESULT`válido, normalmente `S_OK`.
