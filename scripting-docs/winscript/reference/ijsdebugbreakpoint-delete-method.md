# IJsDebugBreakPoint::Delete (M&#233;todo)

## Sintaxis  
  
```  
HRESULT Delete(void);  
```  
  
## Valor devuelto  
  
## Comentarios  
 Devuelve S\_FALSE si se llama en un punto de interrupción eliminado.
