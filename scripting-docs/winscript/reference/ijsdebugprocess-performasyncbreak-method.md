# IJsDebugProcess::PerformAsyncBreak (M&#233;todo)
Coloca el motor de scripts en modo de interrupción, haciendo que se detenga en la siguiente instrucción de scripts.  
  
## Sintaxis  
  
```  
HRESULT PerformAsyncBreak(  
   DWORD threadId  
);  
```  
  
#### Parámetros  
 `threadId`  
 \[in\] El id. del subproceso.
