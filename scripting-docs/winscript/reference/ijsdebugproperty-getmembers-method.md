# IJsDebugProperty::GetMembers (M&#233;todo)
Obtiene los miembros de este objeto.  
  
## Sintaxis  
  
```  
HRESULT GetMembers(  
   JS_PROPERTY_MEMBERS members,  
   IJsEnumDebugProperty **ppEnum  
);  
```  
  
#### Parámetros  
 `members`  
 \[in\] Marcas para especificar qué se incluye en la información sobre el miembro.  
  
 `ppEnum`  
 \[out\] Miembros del objeto.

## Requisitos  
 **Encabezado:** jscript9diag.h
