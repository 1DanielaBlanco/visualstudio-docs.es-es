# IMachineDebugManagerEvents (Interfaz)
Cambios de paso en la lista actual de la aplicación mantenida por el administrador de depuración de equipo.  Esta interfaz se puede utilizar el depurador el IDE para mostrar una lista dinámica de aplicaciones.  
  
 Además de los métodos heredados de `IUnknown`, la interfaz `IMachineDebugManagerEvents` expone los métodos siguientes.  
  
## métodos en el orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IMachineDebugManagerEvents::onAddApplication](../../winscript/reference/imachinedebugmanagerevents-onaddapplication.md)|Controla el evento cuando una aplicación se agrega a la lista actual de la aplicación.|  
|[IMachineDebugManagerEvents::onRemoveApplication](../../winscript/reference/imachinedebugmanagerevents-onremoveapplication.md)|Controla el evento cuando una aplicación se quita de la lista actual de la aplicación.|
