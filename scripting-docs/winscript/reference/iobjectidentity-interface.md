# IObjectIdentity (Interfaz)
Además de los métodos heredados de `IUnknown`, la interfaz `IObjectIdentity` expone los métodos siguientes.  
  
## Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[IObjectIdentity::IsEqualObject](../../winscript/reference/iobjectidentity-isequalobject.md)|Determina si un objeto es igual al objeto actual.|
