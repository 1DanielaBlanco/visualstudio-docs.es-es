# IRemoteDebugApplicationThreadEx (Interfaz)
Representa un subproceso de ejecución dentro de una aplicación determinada.  
  
 Además de los métodos heredados de `IUnknown`, la interfaz `IRemoteDebugApplicationThreadEx` expone los métodos siguientes.  
  
## métodos en el orden de Vtable  
  
|Método|Descripción|  
|------------|-----------------|  
|[IRemoteDebugApplicationThreadEx:EnumGlobalContexts](../../winscript/reference/iremotedebugapplicationthreadex-enumglobalcontexts.md)|Enumera los contextos globales de expresiones para todos los lenguajes que se ejecutan en este subproceso.|
