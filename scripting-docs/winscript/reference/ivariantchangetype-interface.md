# IVariantChangeType (Interfaz)
Permite que el llamador convierte un valor VARIABLE a partir de un tipo a otro.  
  
## Métodos  
 Además de los métodos heredados de `IUnknown`, la interfaz `IVariantChangeType` expone los métodos siguientes.  
  
|Método|Descripción|  
|------------|-----------------|  
|[IVariantChangeType::ChangeType](../../winscript/reference/ivariantchangetype-changetype.md)|Toma un valor variable y crea un nuevo tipo Variant con un tipo especificado.|
