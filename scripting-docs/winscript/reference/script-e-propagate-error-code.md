# SCRIPT_E_PROPAGATE (C&#243;digo de error)
Un error de script se está propagando al llamador, que podría estar en un subproceso diferente.  El host debe pasar el código de error al llamador.  
  
## Sintaxis  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
