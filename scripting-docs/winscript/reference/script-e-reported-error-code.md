# SCRIPT_E_REPORTED (C&#243;digo de error)
El motor de script informa una excepción no controlada al host mediante [IActiveScriptSite::OnScriptError](../../winscript/reference/iactivescriptsite-onscripterror.md).  El host puede omitir este error.  
  
## Sintaxis  
  
```cpp  
SCRIPT_E_REPORTED   0x80020101L  
  
```
