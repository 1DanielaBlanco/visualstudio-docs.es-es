# TEXT_DOC_ATTR (Constantes)
Describe los atributos del documento.  
  
## Sintaxis  
  
```  
typedef DWORD TEXT_DOC_ATTR;  
```  
  
## Constantes  
  
|Constante|Valor|Descripción|  
|---------------|-----------|-----------------|  
|TEXT\_DOC\_ATTR\_READONLY|0x00000001|El documento es de sólo lectura.|  
|TEXT\_DOC\_ATTR\_TYPE\_PRIMARY|0x00000002|El documento es el archivo principal de este árbol de documentos.|  
|TEXT\_DOC\_ATTR\_TYPE\_WORKER|0x00000004|El documento es trabajo.|  
|TEXT\_DOC\_ATTR\_TYPE\_SCRIPT|0x00000008|El documento es un archivo de script.|
