# <a name="handling-expired-subscriptions"></a>Control de suscripciones caducadas

Cuando la suscripción de un usuario ha caducado, esto se indica en el portal mediante una barra roja junto a su nombre. Para que esos suscriptores sigan usando las ventajas de la suscripción de Visual Studio, debe renovar sus licencias. 

![Lista de suscripciones caducadas](_img\expired-subscriptions\expired-list.png) 

Para renovar las suscripciones, póngase en contacto con el distribuidor de licencias por volumen o el distribuidor de Microsoft. Debe proporcionarle el número de contrato y el número de cuenta de compra. El distribuidor de Microsoft o el distribuidor le guiará a través del proceso de renovación o de compra de suscripciones adicionales. Después de realizar el pedido o renovar el contrato, las nuevas suscripciones estarán disponibles y se pueden asignar a los suscriptores.
