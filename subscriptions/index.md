# <a name="visual-studio-subscriptions"></a>Suscripciones de Visual Studio
##  <a name="developer-tools-cloud-services-software-support-and-training"></a>Herramientas de desarrollo, servicios en la nube, software, soporte y aprendizaje

Las suscripciones de Visual Studio son el conjunto de recursos más completo para que pueda crear, implementar y administrar aplicaciones importantes en sus plataformas y dispositivos preferidos, como Windows, iOS, Android y Linux.

# <a name="welcome-to-subscriptions"></a>Bienvenido a las suscripciones

Si es un suscriptor o administra suscripciones para su empresa, puede encontrar información para sacar el máximo provecho de las suscripciones de Visual Studio.
