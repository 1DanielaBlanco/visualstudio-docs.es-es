# <a name="azure-devops-services-benefits-for-visual-studio-subscribers"></a>Ventajas de Azure DevOps Services para suscriptores de Visual Studio

Como suscriptor activo de Visual Studio, puede usar una serie de características de Azure DevOps Services que se incluyen como ventaja en la suscripción. Podrá usar estas mismas características en cada cuenta de Azure DevOps Services de la que sea miembro (porque las haya creado usted mismo o porque otra persona le ha agregado).

Cuando inicie sesión en Azure DevOps Services (ya sea con la identidad que ha usado para activar la suscripción de Visual Studio o con su [identidad alternativa](vs-alternate-identity.md)), se reconocerá automáticamente.

[Registrarse para obtener una cuenta gratuita de Azure DevOps Services](https://visualstudio.microsoft.com/team-services/)

## <a name="eligibility"></a>Elegibilidad

| Nivel de suscripción                                                 |     Canales                                            | Prestación                                                          | ¿Renovable?    |
|--------------------------------------------------------------------|---------------------------------------------------------|------------------------------------------------------------------|---------------|
| Visual Studio Enterprise (estándar, de nube anual, de nube mensual)   | Licencia por volumen, Azure, venta directa, no para reventa<sup>1</sup>, FTE  | Básico, Test Manager, Administración de paquetes, CI/CD autohospedado       |  Sí          |
| Visual Studio Professional (estándar, de nube anual, de nube mensual) | Licencia por volumen, Azure, venta directa                                        | Básico                                                            |  Sí          |
| Visual Studio Test Professional (estándar)                         | Licencia por volumen, venta directa                                              | Básico, Test Manager                                              |  Sí          |
| Plataformas MSDN (estándar)                                          | Licencia por volumen, venta directa                                              | Básico, Test Manager                                              |  Sí          |

<sup>1</sup>  *Incluye: No para reventa (NFR), Microsoft Valued Partner (MVP), Region Director (RD), Visual Studio Industry Partner (VSIP),Microsoft Partner Network (Enterprise), BizSpark, MCT Software & Services Developer. Excluye: MCT Software & Services, Imagine.*

Obtenga más información sobre las características de Azure DevOps Services:

- [Básico](https://visualstudio.microsoft.com/team-services/compare-features/)
- [Test Manager](https://marketplace.visualstudio.com/items?itemName=ms.vss-testmanager-web)
- [Administración de paquetes](https://marketplace.visualstudio.com/items?itemName=ms.feed)

¿No sabe con seguridad qué suscripción usa?  Conéctese a [https://my.visualstudio.com/subscriptions](https://my.visualstudio.com/subscriptions?wt.mc_id=o~msft~docs) para ver todas las suscripciones de Visual Studio que tiene asignadas.
Si no ve todas las suscripciones, es posible que haya una o varias asignadas a una dirección de correo electrónico diferente.  Debe iniciar sesión con esa dirección de correo electrónico para ver esas suscripciones.
